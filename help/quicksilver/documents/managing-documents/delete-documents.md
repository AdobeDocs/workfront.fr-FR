# Supprimer documents

Vous pouvez supprimer les documents que vous téléchargez. Si vous avez l’autorisation de gérer l’accès à des documents spécifiques, vous pouvez également les supprimer.

&#42;Pour connaître le plan, le type de licence ou l’accès dont vous disposez, contactez votre administrateur Workfront.

Pour supprimer un document :

1. Accédez au projet, à la tâche ou au problème qui contient le document, puis sélectionnez **Documents**.
1. Recherchez le document dont vous avez besoin.

1. Cliquez sur le bouton **Supprimer** icon ![](assets/delete.png) au-dessus de la zone Documents.

1. Dans la zone qui s’affiche, cliquez sur **Oui, supprimez-le** pour confirmer.

Un administrateur système ou de groupe peut restaurer un document dans les 30 jours suivant sa suppression, comme décrit dans la section [Restauration des éléments supprimés](../../administration-and-setup/manage-workfront/manage-deleted-items/restore-deleted-items.md).

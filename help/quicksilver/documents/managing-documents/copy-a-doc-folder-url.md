# Copier et partager un lien vers un dossier de document

Vous pouvez copier un lien direct pour n’importe quel dossier de document contenu dans l’un des objets [!DNL Workfront] suivants : [!UICONTROL Programme], [!UICONTROL Portfolio], [!UICONTROL Projet], [!UICONTROL Tâche] ou [!UICONTROL Problème]. Vous ne pouvez pas copier de lien pour les dossiers contenus dans la zone [!UICONTROL Documents] du menu principal, car ces dossiers sont directement liés à votre profil utilisateur et ne peuvent pas être partagés avec d’autres personnes.

>[!NOTE]
>
>Cette fonctionnalité n&#39;est pas disponible dans la zone des nouveaux documents.<br>
>Si votre entreprise utilise l’espace de stockage dans le cloud d’Adobe, la nouvelle zone Documents s’affiche lorsque vous accédez aux documents dans Workfront. Pour plus d’informations sur l’espace de stockage dans le cloud Adobe, consultez [Présentation de l’espace de stockage dans le cloud Adobe](/help/quicksilver/review-and-approve-work/esm-overview.md).

## Copier et partager un lien vers un dossier de document

Pour copier un lien vers un dossier de documents, procédez comme suit :

1. Accédez à l’objet [!DNL Workfront] contenant le dossier de documents.
1. Cliquez sur le menu **[!UICONTROL Plus]**, puis sélectionnez **[!UICONTROL Copier l’URL]**. Vous pouvez partager ce lien pour fournir un accès rapide au dossier. Les utilisateurs doivent au moins disposer d’un accès en lecture seule à l’objet pour afficher le dossier.
   ![Copie de l’url du dossier de documents](assets/copy-doc-folder-url.png)

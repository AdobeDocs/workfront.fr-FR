# Copier et partager un lien vers un dossier de document

Vous pouvez copier un lien direct pour n’importe quel dossier de document contenu dans l’un des éléments suivants : [!DNL Workfront] objets : [!UICONTROL Programme], [!UICONTROL Portfolio], [!UICONTROL Projet], [!UICONTROL Tâche]ou [!UICONTROL Problème]. Vous ne pouvez pas copier de lien pour les dossiers contenus dans le [!UICONTROL Documents] du menu principal, car ces dossiers sont directement liés à votre profil utilisateur et ne peuvent pas être partagés avec d’autres utilisateurs.

## Copier et partager un lien vers un dossier de document

Pour copier un lien vers un dossier de document :

1. Accédez au [!DNL Workfront] contenant le dossier document.
1. Cliquez sur le bouton **[!UICONTROL Plus]** , puis sélectionnez **[!UICONTROL Copier l’URL]**. Vous pouvez partager ce lien pour accéder rapidement au dossier. Les utilisateurs doivent disposer au moins de l’accès Affichage à l’objet pour afficher le dossier.

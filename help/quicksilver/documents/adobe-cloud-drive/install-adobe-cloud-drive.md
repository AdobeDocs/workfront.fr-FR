# Installation d’Adobe Cloud Drive

Pour travailler avec vos projets de stockage dans le cloud Adobe sur votre bureau, installez Adobe Cloud Drive sur votre ordinateur Mac ou Windows.

Pour plus d’informations sur Adobe Cloud Drive et sur ce que vous pouvez en faire, consultez [Présentation d’Adobe Cloud Drive](/help/quicksilver/documents/adobe-cloud-drive/adobe-cloud-drive-overview.md).

## Conditions préalables

Avant d’installer Adobe Cloud Drive, vérifiez les points suivants :

* Votre appareil répond à la configuration requise répertoriée dans la section [&#x200B; Configuration requise &#x200B;](#system-requirements) de cet article.
* Votre organisation utilise une version de Workfront qui prend en charge l’espace de stockage dans le cloud Adobe.

Si votre entreprise gère les installations d’applications pour vous et qu’Adobe Cloud Drive n’est pas disponible au téléchargement, contactez votre administrateur Workfront.

## Configuration requise

**Systèmes d’exploitation :**

* macOS 13.5 ou version ultérieure (Intel et Apple Silicon)
* Windows 11 22H2 (22621.6060) ou plus récent (x64)
* Windows 11 23H2 (22631.6199) ou plus récent (ARM)

**Matériel :**

* 4 Go de RAM minimum
* 500 Mo d’espace disque disponible pour l’application
* Espace supplémentaire pour les fichiers mis en cache (varie selon l’utilisation)

**Network:**

* Connexion Internet pour la configuration initiale et la synchronisation
* Connexion stable recommandée pour les performances

Pour vous assurer qu’Adobe Cloud Drive peut accéder aux services dont il dépend, autorisez les points d’entrée réseau répertoriés dans [Points d’entrée réseau Adobe](https://helpx.adobe.com/in/enterprise/kb/network-endpoints.html).

## Télécharger le programme d’installation

1. Accédez à la [distribution logicielle &#x200B;](https://experience.adobe.com/#/downloads).

1. Connectez-vous avec votre Adobe ID.
1. Dans la liste de téléchargements **Général**, recherchez **Adobe Cloud Drive**.
1. Sélectionnez le programme d’installation de votre système d’exploitation :

   * **macOS :** Adobe Cloud Drive pour Mac (ARM64 ou Intel)
   * **Windows:** Adobe Cloud Drive pour Windows (x64 ou ARM64)

1. Cliquez sur **Télécharger**.

   >[!IMPORTANT]
   >
   >Si le programme d’installation d’Adobe Cloud Drive ne s’affiche pas, il se peut que votre organisation ne dispose pas d’une version de Workfront prenant en charge l’espace de stockage dans le cloud Adobe. Contactez votre administrateur Workfront.

## Installation d’Adobe Cloud Drive

Utilisez la procédure correspondant à votre système d’exploitation.

### Installation sur macOS

1. Ouvrez le fichier `.dmg` téléchargé.
1. Double-cliquez sur l’icône d’installation qui s’affiche.
1. Suivez l’assistant de configuration à l’écran.

1. Lorsque vous y êtes invité, connectez-vous avec votre Adobe ID.

   Si une autre application de bureau Adobe est déjà en cours d’exécution et que vous êtes connecté, Adobe Cloud Drive utilise automatiquement cette session active.

### Installer sous Windows

1. Exécutez le programme d’installation de `.exe` téléchargé.

1. Suivez les invites de l&#39;assistant d&#39;installation.
1. Cliquez sur **Installer** lorsque vous y êtes invité.
1. Lancez **Adobe Cloud Drive** à partir du menu Démarrer.
1. Lorsque vous y êtes invité, connectez-vous avec votre Adobe ID.

## Vérification de l’installation

Après l’installation, Adobe Cloud Drive apparaît comme un lecteur monté sur votre ordinateur :

* **macOS :** dans la barre latérale du Finder sous **Emplacements**, sous la forme **Adobe Cloud Drive**.
* **Windows:** dans l&#39;Explorateur de fichiers sous **Ce PC**, en tant que **Adobe Cloud Drive**.

Vos projets de stockage dans le cloud Adobe doivent apparaître sous la forme de dossiers dans le lecteur.

Pour plus d’informations sur l’utilisation de vos projets sur le lecteur, voir [Utiliser Adobe Cloud Drive](/help/quicksilver/documents/adobe-cloud-drive/use-adobe-cloud-drive.md).

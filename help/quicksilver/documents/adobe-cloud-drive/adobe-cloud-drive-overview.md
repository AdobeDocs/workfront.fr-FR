# Présentation d’Adobe Cloud Drive

Adobe Cloud Drive est une application de bureau qui monte vos projets de stockage dans le cloud Adobe en tant que lecteur sur votre ordinateur Mac ou Windows. Les projets Workfront apparaissent sous la forme de dossiers et vous pouvez ouvrir, créer, modifier et enregistrer des fichiers directement dans les projets Workfront sur l’espace de stockage dans le cloud Adobe, à l’aide de n’importe quelle application de bureau.

Les fichiers d’Adobe Cloud Drive sont identiques à ceux de Workfront et Frame.io. Les modifications que vous apportez à votre bureau sont donc disponibles dans les projets Workfront connectés dans les deux produits.

[Regardez cet aperçu vidéo sur les fonctionnalités disponibles dans Adobe Cloud Drive.](https://video.tv.adobe.com/v/3492492){target="_blank"}

Pour plus d’informations sur le stockage auquel Adobe Cloud Drive se connecte, consultez la [présentation de l’espace de stockage dans le cloud Adobe](/help/quicksilver/review-and-approve-work/esm-overview.md).

## Utilisation d’Adobe Cloud Drive

* Accédez aux projets à partir de l’espace de stockage dans le cloud Adobe sous la forme de dossiers dans le Finder (Mac) ou l’Explorateur de fichiers (Windows).
* Ouvrez, créez et modifiez des fichiers dans les applications Adobe Creative Cloud, Microsoft Office ou toute autre application installée sur votre ordinateur.
* Enregistrez les modifications localement et laissez Adobe Cloud Drive les synchroniser avec les projets sur l’espace de stockage Adobe dans le cloud.
* Marquez les fichiers pour un accès hors ligne lorsque vous voyagez ou travaillez sans connexion réseau. Une fois la connexion restaurée, les mises à jour se synchronisent avec le fichier du projet, puis vous pouvez supprimer les fichiers de votre cache local pour libérer de l’espace disque.
* Collaborez avec les membres de l’équipe sur les mêmes fichiers de projet.

## Qui peut utiliser Adobe Cloud Drive ?

Pour utiliser Adobe Cloud Drive, votre organisation doit se trouver dans le package Workflow Ultimate, avec l’espace de stockage dans le cloud Adobe activé.

Seuls les projets utilisant l’espace de stockage Adobe dans le cloud apparaissent dans le lecteur. Les projets Workfront hérités n’apparaissent pas. Pour plus d’informations sur la différence, voir [Différences entre l’espace de stockage dans le cloud d’Adobe et l’ancien stockage Workfront](/help/quicksilver/review-and-approve-work/esm-overview.md#differences-between-adobe-cloud-storage-and-legacy-workfront-storage).

Si Adobe Cloud Drive n’est pas disponible au téléchargement ou si votre entreprise gère les installations d’applications pour vous, contactez votre administrateur Workfront.

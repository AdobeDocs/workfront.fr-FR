## Filtrage par utilisateur sur le panorama Kanban

Pour filtrer une [!UICONTROL Kanban] panorama :

1. Cliquez sur le bouton **[!UICONTROL Menu Principal]** icon ![](assets/main-menu-icon.png) dans le coin supérieur droit de [!DNL Adobe] Workfront, puis cliquez sur **[!UICONTROL Équipes]**.

1. (Facultatif) Cliquez sur le **[!UICONTROL Equipe de commutation]** icon ![Icône Changer l’équipe](assets/switch-team-icon.png), puis sélectionnez une nouvelle équipe Kanban dans le menu déroulant ou recherchez une équipe dans la barre de recherche.

1. Accédez à [!UICONTROL Kanban] panorama.
1. Cliquez sur le bouton [!UICONTROL Filtrer] menu déroulant sur le côté droit de la [!UICONTROL Kanban] panorama.
1. Sélectionnez un ou plusieurs utilisateurs ou **[!UICONTROL Non attribué]**.

   >[!NOTE]
   >
   >* Les totaux des colonnes ne changent pas en fonction des résultats filtrés. Les totaux des colonnes affichent les totaux de toutes les tâches du panorama. Un maximum de cinquante cartes s’affiche par défaut, mais vous pouvez cliquer sur **[!UICONTROL Afficher plus]** pour afficher d’autres cartes.
   >* Les filtres ne sont pas appliqués à la variable [!UICONTROL Backlog] colonne .

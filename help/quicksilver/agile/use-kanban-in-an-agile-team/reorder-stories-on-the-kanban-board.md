# Réorganiser les articles sur la page [!UICONTROL Kanban] board

Vous pouvez modifier l’ordre dans lequel les articles sont affichés dans les colonnes d’état de la [!UICONTROL Kanban] panorama.

>[!NOTE]
>
>Vous ne pouvez pas modifier l’ordre d’affichage des articles dans le journal d’un événement [!UICONTROL Kanban] l&#39;équipe.

L’ordre dans lequel les articles apparaissent sur la page [!UICONTROL Kanban] le panorama n’indique pas la priorité. Cependant, cela peut affecter la priorité perçue en rendant les articles plus visibles.

Par défaut, les articles s’affichent par ordre alphabétique dans chaque colonne d’état sur la page [!UICONTROL Kanban] panorama.

Lorsque vous réorganisez les colonnes de la variable [!DNL Kanban] panorama, toutes les modifications que vous apportez sont enregistrées, de sorte que les modifications sont conservées la prochaine fois que vous ou un autre utilisateur visualisez la variable [!UICONTROL Kanban] panorama. (Les modifications que vous effectuez ne sont pas annulées lors de l’effacement du cache du navigateur.)

## Réorganiser les articles sur la page [!UICONTROL Kanban] board

Pour modifier l’ordre ou les articles :

1. Cliquez sur le bouton **[!UICONTROL Menu Principal]** icon ![](assets/main-menu-icon.png) dans le coin supérieur droit de [!DNL Adobe] Workfront, puis cliquez sur **[!UICONTROL Équipes]**.

1. (Facultatif) Cliquez sur le **[!UICONTROL Equipe de commutation]** icon ![Icône Changer l’équipe](assets/switch-team-icon.png), puis sélectionnez une nouvelle [!UICONTROL Kanban] dans le menu déroulant ou recherchez une équipe dans la barre de recherche.

1. Accédez au [!UICONTROL Kanban] tableau contenant les articles que vous souhaitez réorganiser.
1. Faites glisser une mosaïque d’article vers l’emplacement vertical souhaité dans une colonne d’état sur le [!UICONTROL Kanban] panorama.

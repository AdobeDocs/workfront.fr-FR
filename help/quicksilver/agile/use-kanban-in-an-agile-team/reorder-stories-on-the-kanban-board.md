# Réorganiser les histoires sur le tableau [!UICONTROL Kanban]

Vous pouvez modifier l’ordre dans lequel les histoires s’affichent dans les colonnes de statut du tableau [!UICONTROL Kanban].

>[!NOTE]
>
>Vous ne pouvez toutefois pas modifier l’ordre d’affichage des histoires dans la liste d’attente pour une équipe [!UICONTROL Kanban].

L’ordre dans lequel les histoires apparaissent sur le tableau [!UICONTROL Kanban] n’est pas révélateur des priorités. Cependant, cela peut affecter la priorité perçue en rendant les histoires plus visibles.

Par défaut, les histoires s’affichent par ordre alphabétique dans chaque colonne de statut sur le tableau [!UICONTROL Kanban].

Lorsque vous réorganisez les colonnes du tableau [!DNL Kanban], toutes les modifications que vous apportez sont enregistrées. Elles sont ainsi conservées pour le prochain affichage du tableau [!UICONTROL Kanban] par vous ou d’autres personnes. (Les modifications que vous effectuez ne sont pas annulées lors de l’effacement du cache du navigateur.)

## Réorganiser les histoires sur le tableau [!UICONTROL Kanban]

Pour modifier l’ordre des histoires, procédez comme suit :

{{step1-to-team}}

1. (Facultatif) Cliquez sur l’icône **[!UICONTROL Changer d’équipe]** ![Icône Changer d’équipe](assets/switch-team-icon.png), puis sélectionnez une nouvelle équipe [!UICONTROL Kanban] dans le menu déroulant ou recherchez une équipe dans la barre de recherche.

1. Accédez au tableau [!UICONTROL Kanban] contenant les histoires que vous souhaitez réorganiser.
1. Faites glisser une mosaïque d’histoire vers l’emplacement vertical souhaité dans une colonne de statut du tableau [!UICONTROL Kanban].

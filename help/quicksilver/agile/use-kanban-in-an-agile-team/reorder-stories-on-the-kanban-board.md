# Réorganiser les articles sur la carte [!UICONTROL Kanban]

Vous pouvez modifier l’ordre dans lequel les articles sont affichés dans les colonnes d’état sur le panorama [!UICONTROL Kanban].

>[!NOTE]
>
>Vous ne pouvez pas modifier l’ordre d’affichage des articles dans le journal d’une équipe [!UICONTROL Kanban].

L&#39;ordre dans lequel les articles apparaissent sur le panorama [!UICONTROL Kanban] n&#39;indique pas la priorité. Cependant, cela peut affecter la priorité perçue en rendant les articles plus visibles.

Par défaut, les articles s’affichent par ordre alphabétique dans chaque colonne d’état sur le panorama [!UICONTROL Kanban].

Lorsque vous réorganisez les colonnes sur le panorama [!DNL Kanban], toutes les modifications que vous apportez sont enregistrées. Les modifications sont donc conservées la prochaine fois que vous ou un autre utilisateur afficherez le panorama [!UICONTROL Kanban]. (Les modifications que vous effectuez ne sont pas annulées lors de l’effacement du cache du navigateur.)

## Réorganiser les articles sur la carte [!UICONTROL Kanban]

Pour modifier l’ordre ou les articles :

1. Cliquez sur l’icône **[!UICONTROL Menu principal]** ![](assets/main-menu-icon.png) dans le coin supérieur droit de [!DNL Adobe] Workfront, puis cliquez sur **[!UICONTROL Équipes]**.

1. (Facultatif) Cliquez sur l’icône **[!UICONTROL Changer d’équipe]** ![Icône Changer d’équipe](assets/switch-team-icon.png), puis sélectionnez une nouvelle équipe [!UICONTROL Kanban] dans le menu déroulant ou recherchez une équipe dans la barre de recherche.

1. Accédez au panorama [!UICONTROL Kanban] qui contient les articles que vous souhaitez réorganiser.
1. Faites glisser une mosaïque d’article vers l’emplacement vertical souhaité dans une colonne d’état sur le panorama [!UICONTROL Kanban].

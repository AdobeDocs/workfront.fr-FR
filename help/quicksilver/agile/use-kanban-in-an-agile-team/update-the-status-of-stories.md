# Mettre à jour l’état des articles sur la variable [!UICONTROL Kanban] board

Vous pouvez modifier l’état d’un article directement à partir du [!UICONTROL Kanban] afin de refléter l&#39;évolution des histoires.

>[!NOTE]
>
>Seuls les états sélectionnés dans la variable [!UICONTROL Story Board] dans la section [!UICONTROL paramètres de l’équipe] sont disponibles sur la [!UICONTROL Kanban] panorama et dans le menu déroulant de l’état. Pour plus d’informations, voir [Configurer Kanban](../../agile/get-started-with-agile-in-workfront/configure-kanban.md)

## Mettre à jour l&#39;état des articles sur le panorama de Kanban

1. Cliquez sur le bouton **[!UICONTROL Menu Principal]** icon ![](assets/main-menu-icon.png) dans le coin supérieur droit de [!DNL Adobe Workfront], puis cliquez sur **[!UICONTROL Équipes]**.

1. (Facultatif) Cliquez sur le **[!UICONTROL Equipe de commutation]** icon ![Icône Changer l’équipe](assets/switch-team-icon.png), puis sélectionnez une nouvelle [!UICONTROL Kanban] dans le menu déroulant ou recherchez une équipe dans la barre de recherche.

1. Accédez au [!UICONTROL Kanban] panorama sur lequel vous souhaitez mettre à jour l’état d’un article.
1. Faites glisser un article d’une colonne d’état sur le [!UICONTROL Kanban] et dans une autre colonne.\
   Une histoire reste dans la [!UICONTROL Terminer] pendant deux semaines après son ajout.\
   ![Déplacer l’histoire d’une autre colonne](assets/agile-storyboard-progress.png)

# Utilisation d’Agile dans un projet

Vous pouvez tirer parti de la fonctionnalité Agile pour votre projet sans les défis administratifs qui accompagnent généralement les pratiques Agile (tels que la gestion de la liste d’attente d’une équipe, la création d’itérations, etc.). Pour plus d’informations, voir [Gérer un projet dans la vue [!UICONTROL Agile]](../../manage-work/projects/manage-projects/manage-projects-in-agile-view.md).

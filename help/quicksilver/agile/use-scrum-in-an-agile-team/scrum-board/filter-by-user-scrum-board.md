## Filtrage par utilisateur sur le tableau de bord

1. Cliquez sur le bouton **[!UICONTROL Menu Principal]** icon ![](assets/main-menu-icon.png) dans le coin supérieur droit de [!DNL Adobe Workfront], puis cliquez sur **[!UICONTROL Équipes]**.

1. (Facultatif) Cliquez sur le **[!UICONTROL Equipe de commutation]** icon ![Icône Changer l’équipe](assets/switch-team-icon.png), puis sélectionnez une nouvelle équipe Scrum dans le menu déroulant ou recherchez une équipe dans la barre de recherche.

1. Accédez à une principale itération.
1. Cliquez sur le bouton [!UICONTROL Filtrer] menu déroulant sur le côté droit de la [!UICONTROL scrum] panorama.
1. Sélectionnez le nom de l’utilisateur ou **[!UICONTROL Non attribué]**.

>[!NOTE]
>
>Les totaux des colonnes ne changent pas en fonction des résultats filtrés. Les totaux des colonnes affichent les totaux de toutes les tâches du panorama.

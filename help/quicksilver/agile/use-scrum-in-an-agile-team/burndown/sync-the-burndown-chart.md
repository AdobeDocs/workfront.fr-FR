# Synchronisation des données dans le graphique de ventilation

Le graphique de charge est synchronisé automatiquement avec les données de votre projet ou de votre itération. Si le graphique de ventilation ne synchronise jamais les données du projet ou de l’itération, vous pouvez le synchroniser manuellement.

## Synchronisation manuelle du graphique de condensation avec les données de votre itération

1. Cliquez sur le bouton **[!UICONTROL Menu Principal]** icon ![](assets/main-menu-icon.png) dans le coin supérieur droit de [!DNL Adobe Workfront], puis cliquez sur **[!UICONTROL Équipes]**.

1. (Facultatif) Cliquez sur le **[!UICONTROL Equipe de commutation]** icon ![Icône Changer l’équipe](assets/switch-team-icon.png), puis sélectionnez une nouvelle équipe Scrum dans le menu déroulant ou recherchez une équipe dans la barre de recherche.

1. Positionnez-vous sur l&#39;itération où vous souhaitez synchroniser manuellement le graphique de charge.
1. Cliquez sur l’icône Synchroniser ![Icône Synchronisation](assets/agile-syncicon.png) en regard du nom de l’itération.

# Afficher une itération

Vous pouvez afficher toutes les itérations d’une équipe donnée ou une itération individuelle. Les itérations affichent des données sur les articles, les problèmes et les documents contenus dans l’itération.

## Afficher les itérations affectées à une équipe donnée

1. Cliquez sur le bouton **[!UICONTROL Menu Principal]** icon ![](assets/main-menu-icon.png) dans le coin supérieur droit de [!DNL Adobe Workfront], puis cliquez sur **[!UICONTROL Équipes]**.

1. (Facultatif) Cliquez sur le **[!UICONTROL Equipe de commutation]** icon ![Icône Changer l’équipe](assets/switch-team-icon.png), puis sélectionnez une nouvelle équipe Scrum dans le menu déroulant ou recherchez une équipe dans la barre de recherche.

1. Dans le panneau de gauche, sélectionnez **[!UICONTROL Itérations]** pour sélectionner une itération spécifique, ou sélectionnez **[!UICONTROL Itération actuelle]**.

   ![Liste d&#39;itération](assets/view-iteration-NWE.png)

   >[!NOTE]
   >
   >**[!UICONTROL Itération actuelle]** s’affiche uniquement dans le panneau de gauche lorsqu’il est affecté au modèle de mise en page et qu’il y a au moins une tâche ou un problème à l’itération. Pour plus d’informations, voir [Personnalisation du panneau de gauche à l’aide d’un modèle de mise en page](/help/quicksilver/administration-and-setup/customize-workfront/use-layout-templates/customize-left-panel.md).


1. (Facultatif) Cliquez sur le nom de l’itération spécifique à afficher.\
   Les articles d’itération s’affichent.

   ![[!UICONTROL Articles en itération]](assets/iteration-stories-list-NWE.png)

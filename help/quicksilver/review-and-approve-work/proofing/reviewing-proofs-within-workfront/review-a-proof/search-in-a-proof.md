# Rechercher du contenu dans une épreuve

Vous pouvez rapidement localiser le texte dans un BAT créé pour les types de documents suivants :

* PDF
* Office (.doc, .docx, .odt)
* Page web statique

>[!NOTE]
>
>Il est possible que les BAT créés avant le 26 avril 2017 ne puissent pas faire l’objet de recherches.

## Rechercher du contenu dans une épreuve

1. Ouvrez le BAT que vous souhaitez rechercher.
1. Dans la barre d&#39;outils au-dessus du BAT, cliquez sur l&#39;icône **Rechercher un document** .

   ![](assets/search-document-icon-search-in-proof-350x129.png)

1. Commencez à saisir le texte à rechercher.

   L’outil de recherche met le texte en surbrillance dans le document au fur et à mesure que vous tapez.

   ![Search_text_underline_as_you_type.png](assets/search-text-highlighted-as-you-type-350x214.png)

1. Terminez de saisir le texte à rechercher, puis cliquez sur les flèches **Up** et **Down** pour parcourir les résultats de recherche dans le BAT.

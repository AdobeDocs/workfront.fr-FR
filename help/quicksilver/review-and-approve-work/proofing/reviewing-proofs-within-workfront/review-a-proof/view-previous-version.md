# Afficher les versions antérieures d’une épreuve dans la visionneuse de relecture

Vous pouvez afficher une version précédente d’un BAT, le cas échéant. Les versions précédentes sont verrouillées par défaut. Vous ne pouvez pas ajouter de commentaires ni modifier une décision sur une version verrouillée.

>[!NOTE]
>
>Les informations décrites dans cet article sont disponibles uniquement avec la visionneuse de BAT Web et uniquement lors de la visionneuse de BAT vidéo ou statique.

## Afficher les versions antérieures d’une épreuve dans la visionneuse de relecture

1. Accédez au projet, à la tâche ou au problème qui contient le document, puis sélectionnez **Documents**.
1. Recherchez le BAT dont vous avez besoin, puis cliquez sur **Ouvrir le BAT**.

1. Dans le coin supérieur gauche de la visionneuse de vérification, cliquez sur le nom du BAT.

   ![phq_viewer_version.png](assets/phq-viewer-version-350x184.png)

1. Dans la liste qui s’affiche, cliquez sur la version que vous souhaitez afficher.
1. (Facultatif) Pour déverrouiller la version si vous souhaitez que les utilisateurs puissent ajouter des commentaires ou modifier une décision, si vous disposez des droits nécessaires, cliquez sur l’icône **Déverrouiller** dans le panneau de gauche, puis cliquez sur **Oui, déverrouiller**. Pour plus d’informations, voir [Verrouiller ou déverrouiller un BAT](../../../../review-and-approve-work/proofing/reviewing-proofs-within-workfront/review-a-proof/lock-or-unlock-proof.md).

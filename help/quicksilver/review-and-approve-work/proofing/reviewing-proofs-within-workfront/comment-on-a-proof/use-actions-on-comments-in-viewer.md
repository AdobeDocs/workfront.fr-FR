# Utiliser des actions sur les commentaires d’une épreuve

Vous pouvez utiliser des actions pour suivre ce qui doit se passer pour chaque thread de commentaires d’une épreuve. Une action est un mot ou une phrase comme « À faire », « Terminé » ou « En cours » que votre équipe d’administration Adobe Workfront configure dans le système pour votre organisation. Les personnes chargées de la révision peuvent ajouter une action à un commentaire afin de fournir des informations sur ce qui a été fait ou ce qui doit encore être fait en réponse au commentaire.

Pour plus d’informations sur l’activation et la configuration des actions par votre équipe d’administration Workfront, consultez .

## Utiliser des actions sur les commentaires

Pour appliquer une action à un commentaire existant dans la visionneuse de relecture :

1. Accédez au projet, à la tâche ou au problème qui contient le document, puis sélectionnez **Documents**.
1. Recherchez l’épreuve dont vous avez besoin, puis cliquez sur **Ouvrir l’épreuve**.

1. Utilisez l’une des méthodes suivantes :

   * Cliquez sur l’icône du drapeau dans le coin inférieur droit du commentaire, puis cliquez sur l’action souhaitée dans le menu déroulant.

     ![Icône d’indicateur Actions](assets/actions-flag-icon-350x198.png)

   * Cliquez sur l’icône **Plus** (trois points horizontaux sur le commentaire), puis cliquez sur l’action souhaitée dans la partie inférieure du menu déroulant qui apparaît.

     ![Actions_on_coments-More_menu.png](assets/actions-on-coments-more-menu-350x347.png)

1. (Facultatif) Si vous changez d’avis, vous pouvez effectuer l’une des opérations suivantes :

   * Cliquez à nouveau sur l’icône du drapeau ou sur l’icône **Plus**, puis cliquez sur **Supprimer l’action**.

   * Répéter l’étape 1 pour appliquer une autre action.

>[!TIP]
>
>Vous pouvez filtrer les commentaires en fonction d’une certaine action. Pour plus d’informations, consultez la section [Rechercher, filtrer et trier les commentaires sur une épreuve](../../../../review-and-approve-work/proofing/reviewing-proofs-within-workfront/comment-on-a-proof/search-filter-sort-comments.md).

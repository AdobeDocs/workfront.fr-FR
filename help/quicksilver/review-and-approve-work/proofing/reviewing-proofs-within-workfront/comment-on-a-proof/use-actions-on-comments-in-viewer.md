# Utiliser des actions sur des commentaires d’épreuve

Vous pouvez utiliser des actions pour effectuer le suivi de ce qui doit se produire sur chaque fil de commentaire sur un BAT. Une action est un mot ou une expression tel que &quot;À faire&quot;, &quot;Terminé&quot; ou &quot;En cours&quot; que votre administrateur Adobe Workfront configure dans le système pour votre organisation. Les réviseurs peuvent ajouter une action à un commentaire pour fournir des informations sur ce qui a été fait ou ce qui doit encore être fait en réponse au commentaire.

Pour plus d’informations sur la manière dont votre administrateur Workfront active et configure des actions, voir .

## Actions sur les commentaires

Pour appliquer une action à un commentaire existant dans la visionneuse de vérification :

1. Accédez au projet, à la tâche ou au problème qui contient le document, puis sélectionnez **Documents**.
1. Recherchez le BAT dont vous avez besoin, puis cliquez sur **Ouvrir le BAT**.

1. Utilisez l’une des méthodes suivantes :

   * Cliquez sur l’icône d’indicateur dans le coin inférieur droit du commentaire, puis cliquez sur l’action de votre choix dans le menu déroulant.

     ![](assets/actions-flag-icon-350x198.png)

   * Cliquez sur l’icône **Plus** (trois points horizontaux sur le commentaire), puis sur l’action souhaitée dans la section inférieure du menu déroulant qui s’affiche.

     ![Actions_on_components-More_menu.png](assets/actions-on-coments-more-menu-350x347.png)

1. (Facultatif) Si vous changez d’avis, vous pouvez effectuer l’une des opérations suivantes :

   * Cliquez à nouveau sur l’icône de l’indicateur ou sur l’icône **Plus**, puis cliquez sur **Supprimer l’action**.

   * Répétez l’étape 1 pour appliquer une autre action.

>[!TIP]
>
>Vous pouvez filtrer les commentaires selon une certaine action. Pour plus d’informations, voir [Rechercher, filtrer et trier les commentaires BAT](../../../../review-and-approve-work/proofing/reviewing-proofs-within-workfront/comment-on-a-proof/search-filter-sort-comments.md).

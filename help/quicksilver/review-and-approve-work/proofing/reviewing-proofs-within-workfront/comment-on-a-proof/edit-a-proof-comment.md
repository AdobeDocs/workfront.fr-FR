# Modifier les commentaires de BAT

Les commentaires ne peuvent pas être modifiés une fois qu’une réponse a été ajoutée au commentaire. Cette restriction conserve l’intégrité du fil de commentaires. Lorsque vous modifiez un commentaire, l’heure à laquelle le commentaire a été fait n’est pas affectée.

## Modifier les commentaires de BAT

Vous pouvez modifier tout commentaire que vous faites sur un BAT. En outre, les utilisateurs suivants peuvent modifier les commentaires d’autres utilisateurs :

* Le propriétaire du BAT
* Le créateur du BAT
* Utilisateurs disposant d’autorisations de profil de superviseur
* Utilisateurs avec un rôle de BAT Auteur ou Modérateur

Pour modifier les commentaires du BAT :

1. Accédez au projet, à la tâche ou au problème qui contient le document, puis sélectionnez **Documents**.
1. Recherchez le BAT dont vous avez besoin, puis cliquez sur **BAT ouvert**.

1. (Conditionnel) Si la zone de commentaires n’est pas ouverte, cliquez sur **Affichage des commentaires** dans le coin supérieur droit.
1. Pointez sur le commentaire à modifier, puis cliquez sur le bouton **Plus** qui s’affiche, puis cliquez sur **Modifier**.

1. ![Edit_a_comment-More_icon.png](assets/edit-a-comment-more-icon.png)

1. Apportez des modifications au commentaire, puis cliquez sur **Post**.

   >[!NOTE]
   >
   >Le libellé &quot;Modifié&quot; apparaît sur le commentaire. Lorsqu’un réviseur vous le demande, votre nom et la date et l’heure des modifications s’affichent. Si vous modifiez le commentaire plusieurs fois, ces informations s’affichent uniquement pour la modification la plus récente. Ce libellé apparaît également au-dessus du commentaire lorsque vous sélectionnez le document dans la zone Documents et affichez le **Mises à jour** dans le résumé.

# Configurer des paramètres de la visionneuse de relecture

Vous pouvez configurer les paramètres suivants pour la visionneuse de relecture web et la visionneuse de relecture de bureau :

* Si les balises et les épingles de commentaire s’affichent sur les épreuves.
* Si les outils de balisage s’affichent dans la partie supérieure de la visionneuse de relecture ou dans un menu déroulant.
* Les notifications par e-mail que vous recevez en tant que réviseur ou réviseuse sur l’épreuve que vous avez ouverte.

* Effacer les données du cache pouvant être enregistrées avec l’épreuve que vous affichez afin de permettre l’affichage dans la visionneuse de contenu tel que des fenêtres contextuelles (qui peuvent être bloquées par les données du cache du navigateur).

## Configurer des paramètres de la visionneuse de relecture

Pour configurer les paramètres de la visionneuse de relecture :

1. Ouvrez la visionneuse de relecture web ou la visionneuse de relecture de bureau de l’une des manières suivantes :

   * Si vous relisez dans Adobe Workfront, accédez à une liste de documents contenant une épreuve à afficher, pointez sur le document, puis cliquez sur **Ouvrir l’épreuve**.
   * Si vous utilisez Workfront Proof, cliquez sur l’icône **Accéder à l’épreuve** correspondant à l’épreuve dans le tableau de bord ou dans une liste Vues ![Accéder à l’épreuve](assets/go-to-proof-blue-icon.png).

1. Si la barre d’outils de gauche ne s’affiche pas, cliquez sur l’icône **Menu**, située dans le coin supérieur gauche de la visionneuse de relecture web.

   ![Icône de menu](assets/menu-icon-in-proofing-viewer-350x228.png)

1. Dans la barre d’outils de gauche, cliquez sur l’icône **Paramètres** ![Icône Paramètres](assets/settings-icon-in-pv.png).

1. Configurez l’un des **Paramètres** suivants qui s’affichent.

   Les paramètres disponibles peuvent varier en fonction du type d’épreuve que vous avez ouverte.

   * **Afficher les balises** (toujours disponible dans la visionneuse de relecture web et la visionneuse de relecture de bureau) : il s’agit des balises de commentaires que les réviseurs et réviseuses ajoutent aux épreuves lorsqu’ils ou elles utilisent les outils de balisage. Si vous les désactivez, vous pouvez toujours les voir lorsque vous cliquez sur un commentaire dans la liste de commentaires.

     Ce paramètre affecte toutes les épreuves que vous ouvrez.

   * **Afficher les épingles** (toujours disponible dans la visionneuse de relecture web et la visionneuse de relecture de bureau) : il s’agit des épingles numérotées que les réviseurs et réviseuses ajoutent aux épreuves lorsqu’ils ou elles utilisent les outils de balisage. Elles indiquent où et dans quel ordre le réviseur ou la réviseuse a ajouté des commentaires. Si vous les désactivez, vous pouvez toujours les voir lorsque vous cliquez sur un commentaire dans la liste de commentaires.

     Ce paramètre affecte toutes les épreuves que vous ouvrez.

   * **Utiliser les outils de balisage développés** (toujours disponible dans la visionneuse de relecture web et la visionneuse de relecture de bureau) : par défaut, les options de l’outil de balisage s’affichent en haut de la visionneuse de relecture.Vous pouvez les configurer pour qu’elles s’affichent dans un menu vertical qui ne s’ouvre que lorsque vous cliquez dessus.

     Ce paramètre s’applique à toutes les épreuves que vous ouvrez.

   * **M’envoyer des notifications par e-mail sur** (toujours disponible dans la visionneuse de relecture web et la visionneuse de relecture de bureau) : cliquez sur l’une des options ci-dessous. Ce paramètre n’affecte que les épreuves que vous avez ouvertes. Pour plus d’informations, voir [Vue d’ensemble des notifications pour les commentaires et les décisions d’épreuves](../../../review-and-approve-work/proofing/proofing-overview/notifications-proof-comments-decisions.md).

   * **Effacer le cache** : efface les données du cache du navigateur qui pourraient être enregistrées avec une épreuve interactive que vous êtes en train de consulter. Cela permet au contenu tel que les fenêtres pop-up (qui peuvent être bloquées par les données du cache du navigateur) de s’afficher dans la visionneuse de relecture de bureau.

     Les données qui sont effacées comprennent le cache HTTP (comme les images qui seront réutilisées après le prochain rafraîchissement de la page) et le cache des données de stockage web (comme les cookies et les données qui identifient les utilisateurs et les utilisatrices).

     Ce paramètre n’affecte que les épreuves que vous avez ouvertes.

# Créer une épreuve interactivenpour un site web ou tout autre contenu web

Vous pouvez générer une nouvelle épreuve interactive ou une nouvelle version d’une épreuve interactive existante pour le contenu web. Il peut s’agir d’un site web ou d’autres types de contenu interactif, tels que des publicités avec diffusion vidéo ou audio en continu, des animations par HTML et des bannières interactives.

Dans une épreuve interactive, les réviseurs et réviseuses peuvent naviguer et interagir comme ils et elles le feraient habituellement avec le site web ou tout autre contenu web.

>[!IMPORTANT]
>
>Assurez-vous que le site web ou le contenu interactif est accessible aux personnes qui vont le consulter. Elles ne peuvent accéder au processus de relecture que si elles peuvent également y accéder sur Internet.

## Créer une épreuve interactivenpour un site web ou tout autre contenu web

1. Accédez au projet, à la tâche ou au problème où vous souhaitez créer une épreuve de site web ou une nouvelle version d’une épreuve existante.
1. Cliquez sur **Documents** dans le panneau de gauche.
1. (Le cas échéant) Si vous créez une épreuve, cliquez sur **Ajouter**, puis cliquez sur **Épreuve** dans le menu qui s’affiche.

1. (Le cas échéant) Sur la page **Nouvelle épreuve** s’affiche, si vous créez une version d’une épreuve existante :

   1. Pointez sur l’épreuve d’URL pour laquelle vous souhaitez créer une version, puis sélectionnez-la en cliquant sur l’arrière-plan bleu clair qui l’entoure.

      ![Select_BAT_by_selection_light_blue_background.png](assets/select-proof-by-selecting-light-blue-background-350x52.png)


   1. Dans le menu déroulant **Ajouter**, cliquez sur **Version** > **Épreuve**.

1. Dans la section **Ajouter des fichiers**, saisissez l’URL du site web que vous souhaitez relire, puis appuyez sur **Entrée**.  Vous pouvez répéter ce processus pour ajouter plusieurs sites web à vérifier.

   ![proof_website.png](assets/proof-website-350x65.png)


   >[!NOTE]
   >
   > L’URL doit comporter moins de 1 000 caractères.

1. Cliquez sur l’URL que vous avez ajoutée.

   ![Cliquez sur URL](assets/click-url-350x137.png)

1. (Facultatif) Si vous souhaitez remplacer le nom de l’épreuve de l’URL du site web par un autre, saisissez un **Nom de l’épreuve**.
1. Sélectionnez **Interactif**, puis cliquez sur **Terminé**.

   >[!NOTE]
   >
   >Si vous ajoutez une nouvelle version à une épreuve d’URL existante, toutes les options configurées sur l’épreuve d’origine ou la version précédente sont conservées dans cette version.

1. Cliquez sur **Créer une épreuve** pour créer une épreuve simple sans processus de révision.\

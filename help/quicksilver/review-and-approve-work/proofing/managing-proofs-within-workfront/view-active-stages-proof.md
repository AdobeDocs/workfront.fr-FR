# Afficher les étapes actives sur une épreuve

Vous pouvez afficher les étapes actives actuelles sur un BAT contenant un workflow automatisé.

&#42;Pour connaître le plan, le rôle ou le profil d’autorisation de BAT dont vous disposez, contactez votre administrateur Workfront ou Workfront Proof.

## Afficher les étapes actives sur une épreuve

1. Passez la souris sur le document qui contient le BAT que vous souhaitez afficher, puis sélectionnez **Workflow de vérification de BAT** pour afficher les étapes du BAT.

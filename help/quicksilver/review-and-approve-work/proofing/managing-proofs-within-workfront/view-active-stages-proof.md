# Afficher les principales étapes sur un BAT

Vous pouvez afficher les principales étapes actuelles sur un BAT contenant un workflow automatisé.

## Exigences d’accès

Vous devez disposer des accès suivants pour effectuer les étapes de cet article :

&#42;Pour connaître le plan, le rôle ou le profil d’autorisation de BAT dont vous disposez, contactez votre administrateur Workfront ou Workfront BAT.

## Afficher les principales étapes sur un BAT

1. Passez la souris sur le document qui contient le BAT à afficher, puis sélectionnez **Workflow de vérification** pour afficher les étapes du BAT.

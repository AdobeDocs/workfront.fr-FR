# Visualiser les étapes actives d’une épreuve

Vous pouvez visualiser les étapes actives actuelles d’une épreuve qui comprend un workflow automatisé.

## Visualiser les étapes actives d’une épreuve

1. Passez la souris sur le document qui contient l’épreuve que vous souhaitez visualiser, puis sélectionnez **Workflow de relecture** pour afficher les étapes de l’épreuve.

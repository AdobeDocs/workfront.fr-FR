# Définir les échéances individuelles d’une personne réviseuse

Vous pouvez définir des échéances différentes pour chaque personne réviseuse sur une épreuve que vous avez déjà créé. Pour ce faire, vous devez créer une étape distincte pour chaque personne réviseuse pour laquelle vous souhaitez définir une échéance individuelle.

## Définir les échéances individuelles d’une personne réviseuse

1. Pointez sur l’épreuve, puis cliquez sur **Workflow de relecture** pour ouvrir la page Workflow.
1. Dans le coin supérieur droit, cliquez sur **Nouvelle étape**.

   ![New_Stage_button.png](assets/new-stage-button.png)

   >[!NOTE]
   >
   >Si vous ne voyez pas l’option Nouvelle étape, vous devez d’abord convertir le workflow en workflow automatisé.

1. Ajoutez des informations sur l’étape, notamment le nom de la personne réviseuse et la date d’échéance pour la personne réviseuse.
1. Répétez les deux étapes précédentes si nécessaire pour ajouter les autres personnes réviseuses ayant besoin d’une échéance spécifique.

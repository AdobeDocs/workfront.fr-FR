# Définition des échéances individuelles des validants

Vous pouvez définir des échéances différentes pour différents opérateurs validants sur un BAT que vous avez déjà créé. Cela nécessite la création d’une étape distincte pour chaque réviseur pour lequel vous souhaitez définir une échéance individuelle.

## Définition des échéances individuelles des validants

1. Passez la souris sur le BAT, puis cliquez sur **Workflow de vérification** pour ouvrir la page Processus .
1. Dans le coin supérieur droit, cliquez sur **Nouvelle étape**

   ![New_Stage_button.png](assets/new-stage-button.png)

   >[!NOTE]
   >
   >Si vous ne voyez pas la nouvelle option d’étape, vous devez d’abord convertir le workflow en workflow automatisé.

1. Ajoutez des informations pour l&#39;étape, notamment le nom du validant et la date limite de sélection du validant.
1. Répétez les deux étapes précédentes si nécessaire pour ajouter les autres opérateurs validants ayant besoin de leur propre délai.

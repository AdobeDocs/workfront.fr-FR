## Afficher la liste de toutes les versions de BAT

1. Dans la liste Document, sélectionnez le BAT.
1. Dans le résumé, faites défiler l’écran jusqu’à la section **Toutes les versions** . Ici, vous pouvez visualiser toutes les versions du BAT.

   ![](assets/copy-of-versions-350x173.png)

## Aperçu d’une version précédente du BAT

Les fichiers qui ne peuvent pas être prévisualisés (tels que XLSX et DOC) sont téléchargés.

1. Dans la liste des documents, sélectionnez un BAT.
1. Dans le résumé, faites défiler l’écran jusqu’à **Versions**, cliquez sur le menu **Plus** , puis sélectionnez **Aperçu**.

   ![](assets/proof-versions-in-summary-350x167.png)

## Afficher une version précédente du BAT

Tout utilisateur de Workfront ayant accès à l’affichage du document peut afficher les versions antérieures d’un document révisé. Il n’est pas nécessaire que l’utilisateur dispose d’une licence de vérification.

1. Accédez au projet, à la tâche ou au problème qui contient le document, puis sélectionnez **Documents**.
1. Trouvez le bon à tirer dont vous avez besoin.
1. Dans le résumé, faites défiler l’écran jusqu’à **Versions**, cliquez sur le menu **Plus** , puis sélectionnez **Ouvrir le BAT**.

   ![](assets/proof-versions-in-summary-350x167.png)

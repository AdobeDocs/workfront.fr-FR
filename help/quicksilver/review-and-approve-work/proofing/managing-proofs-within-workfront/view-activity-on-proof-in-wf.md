# Afficher l’activité d’une épreuve dans Adobe Workfront

Vous pouvez afficher l’activité de l’épreuve. L’activité comprend les modifications récentes apportées à l’épreuve, notamment la date et la description de l’activité ainsi que l’utilisateur ou l’utilisatrice qui a effectué l’action.

## Afficher l’activité d’une épreuve dans Adobe Workfront

1. Pointez sur le document qui contient l’épreuve, puis sélectionnez **Détails du document**.
1. Dans le menu de navigation de gauche, sélectionnez **Activités de relecture**.

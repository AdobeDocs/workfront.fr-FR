# Vue d’ensemble des différences entre la visionneuse de relecture web et la visionneuse de relecture de bureau

Adobe Workfront fournit deux visionneuses de relecture :

* **Visionneuse de relecture web :** conçue principalement pour la relecture des fichiers statiques et vidéo. Elle s’exécute dans Google Chrome, Firefox ou Safari.
* **Visionneuse de relecture de bureau :** conçu pour la relecture des fichiers interactifs, ainsi que des fichiers vidéo et statiques.Elle s’exécute en tant qu’application autonome sur votre poste de travail. Pour plus d’informations, voir [Comprendre la visionneuse de relecture de bureau](../../../workfront-proof/wp-work-proofsfiles/review-proofs-dpv/destop-proofing-viewer.md)

* Si votre entreprise ne peut pas utiliser l’application de la visionneuse de relecture de bureau pour des raisons de sécurité, votre administrateur ou administratrice Workfront peut configurer votre système afin que vous puissiez réviser le contenu interactif, regroupé dans un fichier d’archive ZIP, dans la visionneuse de relecture web. Pour plus d’informations, voir  [Configurer la relecture de contenu interactif dans la visionneuse de relecture web](../../../review-and-approve-work/proofing/reviewing-proofs-within-workfront/review-a-proof/interactive-content-in-web-viewer.md).

La liste suivante peut vous aider à comprendre quelle visionneuse de relecture utiliser pour réviser certains types de contenu :

* **Contenu web interactif - URL** : si vous créez une épreuve pour le contenu web à l’aide d’une URL et que vous souhaitez relire le contenu de manière interactive, vous devez utiliser la visionneuse de relecture de bureau.
* **Contenu web interactif - Fichier ZIP** : si vous créez une épreuve pour le contenu web à l’aide d’un fichier ZIP, vous pouvez utiliser soit la visionneuse de relecture web (avec certaines restrictions), soit celle de bureau. Pour plus d’informations sur les restrictions liées à l’utilisation de la visionneuse de relecture web pour le contenu interactif, voir [Configurer la relecture de contenu interactif dans la visionneuse de relecture web](../../../review-and-approve-work/proofing/reviewing-proofs-within-workfront/review-a-proof/interactive-content-in-web-viewer.md).

* **Contenu statique et vidéo** : si vous créez une épreuve contenant du contenu statique, vous pouvez utiliser soit la visionneuse de relecture web, soit celle de bureau.

## Épreuves statiques

| **Fonctionnalité** | **Visionneuse de relecture web** | **Visionneuse de relecture de bureau** |
|---|---|---|
| Ouvrir les épreuves statiques | ✓ | ✓&#42; |
| Vues unique, magazine et continue | ✓ | ✓&#42; |
| Panoramique | ✓ | ✓&#42; |
| Zoom | ✓ | ✓&#42; |
| Faire pivoter | ✓ | ✓&#42; |
| Outil de mesure | ✓ (définissez une zone de taille personnalisée) | ✓&#42; |
| Vue Miniature | ✓ | ✓&#42; |
| Navigateur d’épreuves statiques | ✓ | ✓&#42; |
| Recherche de documents | ✓ | ✓&#42; |
| Publier un commentaire sur plusieurs pages | ✓ (disponible sur toutes les vues) | ✓&#42; (disponible sur toutes les vues) |
| Raccourcis d’épreuve statique avancés | ✓ (pour plus d’informations, voir [Raccourcis clavier dans la visionneuse de relecture](../../../review-and-approve-work/proofing/reviewing-proofs-within-workfront/review-a-proof/keyboard-shortcuts-proof.md)) | ✓&#42;(pour plus d’informations, voir [Raccourcis clavier dans la visionneuse de relecture](../../../review-and-approve-work/proofing/reviewing-proofs-within-workfront/review-a-proof/keyboard-shortcuts-proof.md)) |

{style="table-layout:auto"}

&#42; Cette fonctionnalité fonctionne uniquement si votre administrateur ou administratrice Workfront a configuré la visionneuse de relecture de bureau en tant que visionneuse par défaut pour toutes les épreuves.

| **Fonctionnalité**  | **Visionneuse de relecture web** | **Visionneuse de relecture de bureau** |
|---|---|---|
| Ouvrir des épreuves vidéo | ✓ | ✓&#42; |
| Mise en mémoire tampon | ✓ | ✓&#42; |
| Révision à l’aide du temps | ✓ | ✓&#42; |
| Révision à l’aide d’images ou du code temporel | ✓ | ✓&#42; |
| Révision plus rapide ou plus lente | ✓ | ✓&#42; |
| Régulation des volumes | ✓ | ✓&#42;  |
| Mode Plein écran | ✓ | ✓&#42;  |
| Commentaires sur plage | ✓ | ✓&#42; |
| Épreuves vidéo en boucle (les vidéos se terminent et démarrent automatiquement) | ✓ | ✓&#42;  |
| Raccourcis vidéo avancés | ✓ (pour plus d’informations, voir [Raccourcis clavier dans la visionneuse de relecture](../../../review-and-approve-work/proofing/reviewing-proofs-within-workfront/review-a-proof/keyboard-shortcuts-proof.md)) | ✓&#42;  |

{style="table-layout:auto"}

&#42; Cette fonctionnalité fonctionne uniquement si la visionneuse de relecture de bureau est configurée comme visionneuse par défaut pour toutes les épreuves.

## Épreuves interactives

| **Fonctionnalité** | **Visionneuse de relecture web** | **Visionneuse de relecture de bureau** |
|---|---|---|
| Ouvrir les épreuves interactives créées à partir de contenu compressé dans un fichier ZIP | ✓ | ✓ (recommandé) |
| Ouvrir une épreuve interactive créée à partir d’une URL | Non pris en charge | ✓ |
| Afficher des épreuves interactives (créées à partir de contenu compressé dans un fichier ZIP) dans diverses tailles d’écran | ✓ | ✓ |
| Afficher des épreuves interactives (créées à partir de contenu compressé dans un fichier ZIP) pour divers appareils | Non pris en charge | ✓ |
| Réviser les sites (HTTP) non sécurisés | Non pris en charge | ✓ |
| Réviser les sites protégés des iFrame (sites protégés contre l’affichage dans un iFrame) | Non pris en charge | ✓ |

# Gestion des dossiers et de leur contenu dans [!DNL Workfront Proof]

>[!IMPORTANT]
>
>Cet article fait référence à la fonctionnalité du produit autonome [!DNL Workfront Proof]. Pour plus d&#39;informations sur la vérification à l&#39;intérieur de [!DNL Adobe Workfront], voir [Proofing](../../../review-and-approve-work/proofing/proofing.md).

L’un des avantages de l’organisation de vos projets et BAT dans des dossiers est la possibilité de les gérer sur la page [!UICONTROL Détails du dossier]. Cette page est un centre de contrôle pour votre projet. Vous pouvez ainsi facilement gérer des bons à tirer et des fichiers individuels et effectuer des actions en bloc.

## Regroupement D’Éléments Pour Que Les Réviseurs Puissent Y Travailler Ensemble

Vous pouvez utiliser des dossiers pour regrouper les BAT sur lesquels les réviseurs doivent travailler ensemble. Lorsqu’un réviseur ouvre l’un des BAT du dossier pour lancer la visionneuse de BAT, tous les autres BAT du dossier sont disponibles. Le réviseur peut travailler sur n’importe quel BAT du dossier sans quitter la visionneuse de BAT. Pour plus d’informations, voir [Utilisation de plusieurs bons à tirer dans la visionneuse de correctifs](../../../workfront-proof/wp-work-proofsfiles/review-proofs-wpv/work-with-multiple-proofs.md).

## Ajout de nouveaux éléments à un dossier

Lorsque vous créez un BAT ou téléchargez un fichier, vous pouvez sélectionner le dossier dans lequel vous souhaitez le stocker dans la section [!UICONTROL Organiser] de la page [!UICONTROL Nouveau BAT] ou de la page [!UICONTROL Nouveau fichier].

Pour plus d’informations, voir [Générer des bons à tirer dans [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/create-proofs-and-files/generate-proofs.md) ou [Télécharger des fichiers et du contenu web vers [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/create-proofs-and-files/upload-files-web-content.md).

>[!NOTE]
>
>Si vous affichez actuellement le dossier auquel vous souhaitez ajouter un nouvel élément, ce dossier apparaît automatiquement dans la section [!UICONTROL Organiser] lorsque vous ouvrez la page Nouveau BAT ou la page [!UICONTROL Nouveau fichier].

## Ajout d’éléments existants à un dossier

Vous pouvez ajouter un fichier ou un BAT existant à un dossier.

Pour plus d’informations, voir [Gestion des fichiers dans [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/manage-your-work/manage-files.md) ou [Gestion des détails du bon à tirer dans Workfront Proof](../../../workfront-proof/wp-work-proofsfiles/manage-your-work/manage-proof-details.md).

1. (Le cas échéant) Effectuez l’une des opérations suivantes :

   * Si vous ajoutez un BAT à un dossier, accédez à la page [!UICONTROL Détails du BAT] comme décrit dans [[!UICONTROL Gérer les détails du BAT] dans [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/manage-your-work/manage-proof-details.md)
   * Si vous ajoutez un fichier à un dossier, accédez à la page Détails du fichier comme décrit dans la section [Gérer les fichiers dans [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/manage-your-work/manage-files.md).&quot;

1. Cliquez sur le nom du **Dossier** où se trouve le fichier ou le BAT, puis cliquez sur le nom du dossier dans lequel vous souhaitez le déplacer.\
   ![Gestion_dossier_items-adding_items_to_a_folder.png](assets/managing-folder-items-adding-items-to-a-folder-350x121.png)

## Déplacement d’éléments d’un dossier

1. (Le cas échéant) Effectuez l’une des opérations suivantes :

   * Si vous sortez un BAT d’un dossier, accédez à la page [!UICONTROL Détails du BAT] comme décrit dans la section [[!UICONTROL Gérer les détails du BAT] dans [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/manage-your-work/manage-proof-details.md)\

     Ou\
      Si vous déplacez un fichier hors d’un dossier, accédez à la page Détails du fichier comme décrit dans la section [Gérer les fichiers dans [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/manage-your-work/manage-files.md).&quot;

1. Cliquez sur le nom du **Dossier** où se trouve le fichier ou le BAT, puis cliquez sur **[!UICONTROL Déplacer vers]** > **[!UICONTROL (Aucun dossier sélectionné)]**.\
   ![Gestion des dossiers_éléments-déplacement_éléments_out_des_dossiers.png{1](assets/managing-folder-items-moving-items-out-of-folders-350x123.png)

## Gestion d’un dossier et de ses éléments sur la page [!UICONTROL Détails du dossier]

Vous pouvez facilement gérer vos fichiers à partir de la page [!UICONTROL Détails du dossier].

>[!NOTE]
>
>Si vous n’êtes pas le créateur ou le propriétaire du fichier, les options disponibles dépendent de votre utilisateur [Profils d’autorisations de BAT dans [!DNL Workfront Proof]](../../../workfront-proof/wp-acct-admin/account-settings/proof-perm-profiles-in-wp.md).

1. Dans l’onglet **[!UICONTROL Dossiers]** du menu de navigation de gauche, cliquez sur n’importe quel dossier pour ouvrir la page [!UICONTROL Détails du dossier].
1. Si le dossier contient uniquement des BAT archivés, un message s’affiche en jaune en haut de la page. Si vous cliquez sur le lien contenu dans le message, la vue du dossier passe à la vue [!UICONTROL BAT archivé].
1. Cliquez sur **[!UICONTROL Détails du dossier]**, puis effectuez l’une des opérations suivantes avec le dossier à l’aide des options qui s’affichent :

   * Ajoutez ou modifiez l’image miniature du dossier.
   * Affichez ou modifiez le **Description**, le **dossier parent**, le **propriétaire**, le **client** ou le **projet** associé au dossier.

   * Rendez le dossier **Privé**.
   * Voir quand le dossier a été **Créé**.
   * Voir le nom du **créateur** du dossier.

1. (Facultatif) Pour modifier les informations associées à un élément du dossier, cliquez sur l’icône **[!UICONTROL Plus]** située à droite de l’élément, puis utilisez l’une des options qui s’affichent.\
   ![](assets/more-button-small.png)\
   Les options qui s’affichent dépendent si l’élément est un BAT ou un fichier.

   * **[!UICONTROL Afficher les détails du BAT]** ou **[!UICONTROL Afficher les détails du fichier]** : ouvre la page Détails du BAT ou Détails du fichier . Pour plus d’informations, voir [Gestion des fichiers dans [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/manage-your-work/manage-files.md) ou [Gestion des détails du bon à tirer dans [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/manage-your-work/manage-proof-details.md).

   * **[!UICONTROL Partager]** : vous permet de partager le fichier avec d’autres personnes. Pour plus d’informations, voir [Partage de fichiers dans [!UICONTROL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/share-proofs-and-files/share-files.md).

   * **[!UICONTROL Message]** : envoyez un email aux personnes avec lesquelles le BAT a été partagé.
   * **[!UICONTROL Nouvelle version]** : créez une nouvelle version du BAT.
   * **[!UICONTROL Copier]** : copiez le BAT avec tout commentaire de vérification sous la forme d’une nouvelle version d’un BAT existant ou d’un nouveau BAT.
   * **[!UICONTROL Télécharger l’original]** : vous permet de télécharger le fichier d’origine. Pour plus d’informations, voir [Téléchargement de fichiers stockés dans [!DNL Workfront Proof]](../../../workfront-proof/wp-work-proofsfiles/manage-your-work/download-files-stored.md).

   * **[!UICONTROL Déléguer la propriété]** : déléguez la propriété du BAT à un autre utilisateur.
   * **[!UICONTROL Partager les liens du BAT]** : envoyez un email contenant un lien vers le BAT.
   * **[!UICONTROL Imprimer les commentaires]** : imprime les commentaires que les réviseurs ont faits sur le BAT.
   * **[!UICONTROL Résumé Excel]** : créez un fichier Excel contenant la synthèse d’impression du BAT.
   * **[!UICONTROL Verrouiller]** : verrouillez le BAT pour qu’aucun autre utilisateur ne puisse l’ouvrir.
   * **[!UICONTROL Supprimer]** : supprime le fichier de [!DNL Workfront] BAT.

1. (Facultatif) Dans la section **[!UICONTROL Partagé avec]**, pour envoyer un rappel à un réviseur sur le BAT, cliquez sur l&#39;icône **[!UICONTROL Plus]** à droite du nom de l&#39;utilisateur, puis cliquez sur **[!UICONTROL Message]**.\
   Vous pouvez cliquer sur **[!UICONTROL Supprimer]** pour annuler le partage du dossier avec la personne.\
   Les propriétaires et les créateurs de dossiers apparaissent toujours dans cette liste et ne peuvent pas être supprimés. Si un utilisateur reprend la propriété d’un dossier, il apparaît automatiquement et ne peut pas être supprimé.\
   Vous pouvez partager un dossier avec des utilisateurs individuels et des sociétés partenaires. Si le dossier est partagé avec une société Partner, vous pouvez afficher la liste complète des destinataires en cliquant sur l’icône [!UICONTROL information] qui s’affiche lorsque vous pointez sur la ligne. Pour plus d’informations, voir [Comptes partenaires.](https://support.workfront.com/hc/en-us/sections/115000912107-Partner-accounts)

1. (Facultatif) Modifiez les autorisations du gestionnaire sur les dossiers privés en cliquant sur la ligne de cette personne sous **[!UICONTROL Autoriser sur le dossier]**, puis en indiquant si vous souhaitez que la personne puisse créer, modifier et supprimer des éléments de ce dossier. **REMARQUE** Cette option est disponible uniquement pour les gestionnaires du même compte que le propriétaire du dossier qui sont ajoutés à des dossiers privés. Les superviseurs, les administrateurs et les administrateurs de facturation disposent de droits de modification sur tous les dossiers du compte et les observateurs ne peuvent créer aucun élément. Pour plus d’informations sur les autorisations, voir [ Profils d’autorisations de BAT dans  [!DNL Workfront Proof]](../../../workfront-proof/wp-acct-admin/account-settings/proof-perm-profiles-in-wp.md).

1. (Facultatif) Cliquez sur **[!UICONTROL Activité]** pour afficher toutes les actions effectuées dans le dossier par ordre chronologique.
1. (Facultatif) Cliquez sur la section **[!UICONTROL Messages]** pour afficher les messages qui ont été envoyés aux utilisateurs du dossier.\
   Lorsque vous passez la souris sur un message, vous pouvez cliquer sur l’icône **i** à droite pour afficher le message complet.\
   ![](assets/messages-1-350x74.png)\
   Une fenêtre contextuelle avec le message électronique complet s’affiche :\
   ![](assets/messages-2-350x252.png)

## Gestion des sous-dossiers

Vous pouvez gérer les sous-dossiers à partir de la page Détails des dossiers .

1. Dans l’onglet **[!UICONTROL Dossiers]** du menu de navigation de gauche, cliquez sur le nom du dossier parent qui contient le sous-dossier pour ouvrir la page **[!UICONTROL Détails du dossier]**.

1. Cliquez sur le menu **[!UICONTROL Plus]** à droite du sous-dossier.\
   ![More_button_small.png](assets/more-button-small.png)

1. Cliquez sur l’une des options suivantes :

   * **[!UICONTROL Afficher les détails du dossier]** : ouvre la page [!UICONTROL Détails du dossier].
   * **[!UICONTROL Nouveau BAT]** : vous permet d’ajouter un nouveau BAT au sous-dossier.
   * **[!UICONTROL Télécharger le fichier]** : vous permet de télécharger un fichier dans le sous-dossier.
   * **[!UICONTROL Nouveau sous-dossier]** : crée un nouveau sous-dossier dans le sous-dossier.
   * **[!UICONTROL Partager]** : vous permet de partager le sous-dossier avec d’autres utilisateurs.
   * **[!UICONTROL Résumé Excel]** : vous permet de demander un résumé Excel du sous-dossier.
   * **[!UICONTROL Supprimer]** : permet de supprimer le sous-dossier.

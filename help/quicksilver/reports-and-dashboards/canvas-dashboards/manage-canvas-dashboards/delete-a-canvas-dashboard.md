## Supprimer un tableau de bord

>[!WARNING]
>
> Une fois qu&#39;un tableau de bord est supprimé, le tableau de bord et tous ses rapports et/ou visualisations personnalisés ne peuvent pas être récupérés.<br>
> Si vous supprimez un tableau de bord contenant un rapport classique, ce dernier n’est pas supprimé.

{{step1-to-dashboards}}

1. Dans le panneau de gauche, cliquez sur **Tableaux de bord des zones de travail**.

1. Sur la page **Tableaux de bord Canvas**, sélectionnez le tableau de bord que vous souhaitez supprimer.

1. Dans le coin supérieur droit, sélectionnez l&#39;icône **Plus** ![Plus](assets/more-icon.png), puis **Supprimer le tableau de bord**.
   ![Supprimer le tableau de bord](assets/delete-dashboard.png)

1. Dans la boîte de dialogue **Supprimer le tableau de bord**, cochez la case **Je confirme que je souhaite supprimer ce tableau de bord**.

1. Cliquez sur **Supprimer**.

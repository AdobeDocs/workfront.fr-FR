# Améliorations de la gestion des documents pour le quatrième trimestre 2024

Cette page décrit toutes les améliorations apportées à la gestion des documents dans l’environnement Aperçu depuis la version du quatrième trimestre 2024. Ces améliorations seront disponibles dans l’environnement de production avec la version du quatrième trimestre 2024.

Pour une liste de tous les changements disponibles à ce stade du cycle de publication de la version du quatrième trimestre 2024, voir [Vue d’ensemble de la version du quatrième trimestre 2024](/help/quicksilver/product-announcements/product-releases/24-q4-release-activity/24-q4-release-overview.md).

>[!IMPORTANT]
>
>Les fonctionnalités répertoriées sur cette page font partie d’une version par étapes et ne sont disponibles que pour des clients spécifiques.

## Afficher le statut de décision du document directement dans la liste des documents

>[!NOTE]
>
>~~Version préliminaire : 3 octobre 2024~~ ; production pour tous les clients : avec la version 24.10 (17 octobre 2024)

Vous pouvez désormais afficher le statut de décision d’un document directement dans la liste des documents.

Pour plus d&#39;informations, voir [Présentation de l&#39;état de décision du document](/help/quicksilver/review-and-approve-work/document-reviews-and-approvals/manage-document-approvals/document-approval-status.md).

## Ajout rapide de réviseurs et d’approbateurs précédents aux nouvelles versions de document

>[!NOTE]
>
>Version préliminaire : 3 octobre 2024 ; Production pour tous les clients : avec la version 24.10 (17 octobre 2024)

Vous pouvez désormais ajouter rapidement des réviseurs et des approbateurs à partir de versions de documents précédentes.

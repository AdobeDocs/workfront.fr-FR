# Vue d’ensemble de l’activité Version R1

Cette page fournit des informations sur les fonctionnalités incluses dans la version R1. 

La version R1 est actuellement disponible pour le test dans Aperçu ; les versions R1 de l’environnement de production sont déployées par étapes les soirs des 26 et 27 avril 2017 (MDT).

La version R1 permet aux utilisateurs, aux professionnels de la connaissance et aux administrateurs de mieux gérer leur travail. Avec R1, les administrateurs peuvent personnaliser Workfront pour mieux correspondre à la terminologie de l’entreprise ; les professionnels du savoir peuvent tirer parti des rapports d’utilisation pour mieux comprendre comment les projets effectuent le suivi par rapport aux heures planifiées et budgétées ; les utilisateurs peuvent s’engager dans Workfront directement depuis Outlook 365 et être plus productifs lors de l’utilisation de l’application mobile Workfront ; et bien plus encore.

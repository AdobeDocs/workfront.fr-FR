# Vue d’ensemble de la version 22.1

Cette page fournit des informations sur les fonctionnalités incluses dans la version 22.1. Toutes les fonctionnalités sont disponibles dans la nouvelle expérience Adobe Workfront. Certaines fonctionnalités sont également disponibles dans Adobe Workfront Classic. Toutefois, [Workfront Classic sera abandonné en mars 2022](https://one.workfront.com/s/new-workfront-experience) et atteindra sa date de fin de vie en juillet 2022.

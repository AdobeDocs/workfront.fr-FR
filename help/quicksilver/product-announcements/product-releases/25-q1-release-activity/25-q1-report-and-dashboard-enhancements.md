# Améliorations apportées au rapport et au tableau de bord du premier trimestre 2025

Cette page décrit toutes les améliorations apportées aux rapports et aux tableaux de bord avec la version du premier trimestre 2025 dans l’environnement Aperçu. Ces améliorations seront rendues disponibles comme indiqué, dans l’environnement de production.

Pour obtenir la liste de toutes les modifications disponibles à ce stade du cycle de publication de la version du premier trimestre 2025, consultez l’article [Vue d’ensemble de la version du premier trimestre 2025](/help/quicksilver/product-announcements/product-releases/25-q1-release-activity/25-q1-release-overview.md).

## Nouvelles entités disponibles dans Data Connect

>[!NOTE]
>
>Version de prévisualisation : jeudi 15 janvier 2025. Version de production pour l’ensemble des clientes et clients : jeudi 15 janvier 2025.

Nous avons ajouté la prise en charge des entités suivantes dans Data Connect :

* Utilisateur de l’équipe de projet
* Rôle Utilisateur de l’équipe de projet
* Heures budgétées à déclarer
* Nombre de statistiques de vue de rapport
* Gestionnaire des ressources
* Remarque Rich Text

Nous avons également apporté notre soutien aux entités suivantes :

* Profil professionnel
* Règles métier
* Emplacement
* Catégorie de ressources non liée à la main-d’œuvre
* Emplacement de l’utilisateur

Pour plus d&#39;informations, voir [Dictionnaire de données Workfront Data Connect](/help/quicksilver/reports-and-dashboards/data-lake/data-dictionary.md).

## Limite de 25 rapports, pages externes ou calendriers dans les tableaux de bord

>[!NOTE]
>
>Version d’aperçu : 16 décembre 2024 ; version de production pour tous les clients : avec la version 25.1 (16 janvier 2025)

Pour conserver les performances du tableau de bord, nous avons limité le nombre total de rapports, de pages externes ou de calendriers pouvant être placés dans un tableau de bord. Lors de la création d’un nouveau tableau de bord, un maximum de 25 éléments peut être ajouté.

Les tableaux de bord existants qui dépassent cette limite affichent un avertissement indiquant que seuls les 25 éléments les plus élevés sont affichés. Une fois modifié, le tableau de bord ne peut pas être enregistré tant que le nombre d&#39;éléments inclus n&#39;est pas réduit à 25 ou moins.

Pour plus d&#39;informations, voir [Créer un tableau de bord](/help/quicksilver/reports-and-dashboards/dashboards/creating-and-managing-dashboards/create-dashboard.md).

## Bouton de création de compte de premier lecteur pour Data Connect

>[!NOTE]
>
>Version de production pour tous les clients : 14 novembre 2024

Les administrateurs qui accèdent à Data Connect pour la première fois ont désormais la possibilité de créer un nouveau compte de lecteur de Snowflake en cliquant sur un seul bouton. Le processus prend quelques minutes, mais ne nécessite aucune action supplémentaire.

Pour plus d&#39;informations sur la configuration de Data Connect, voir [Créer un compte de lecteur pour Snowflake](/help/quicksilver/reports-and-dashboards/data-lake/create-a-reader-account.md).

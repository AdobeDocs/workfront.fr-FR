# Améliorations apportées aux panoramas (1er trimestre 2025)

Cette page décrit toutes les améliorations apportées aux tableaux dans l’environnement Aperçu avec la version du premier trimestre 2025. Ces améliorations seront rendues disponibles comme indiqué, dans l’environnement de production.

Pour obtenir la liste de toutes les modifications disponibles à ce stade du cycle de publication de la version du premier trimestre 2025, consultez l’article [Vue d’ensemble de la version du premier trimestre 2025](/help/quicksilver/product-announcements/product-releases/25-q1-release-activity/25-q1-release-overview.md).

## Modification du propriétaire d’un forum

>[!NOTE]
>
>Version d’aperçu : 18 décembre 2024 ; version de production pour tous les clients : avec la version 25.1 (janvier 2025)

Le créateur d’un forum en est le propriétaire par défaut. Le propriétaire du forum est la seule personne qui peut supprimer ce forum ou mettre à jour ses filtres dans le panneau Configurer.

Une fonctionnalité a été ajoutée pour permettre aux administrateurs système de Workfront de modifier le propriétaire d’un forum. Le propriétaire actuel d&#39;un forum peut également modifier le propriétaire de ce forum spécifique. Cette fonctionnalité est disponible sur les tableaux de base, rétrospectifs et Kanban, mais pas sur les tableaux dynamiques.

Auparavant, il était impossible de changer le propriétaire de la commission sans intervention en aval.

Pour plus d’informations, consultez [Ajouter ou supprimer des personnes membres d’un panorama](/help/quicksilver/agile/get-started-with-boards/add-members-to-board.md).

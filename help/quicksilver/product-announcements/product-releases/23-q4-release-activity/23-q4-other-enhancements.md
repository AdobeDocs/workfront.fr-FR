# Autres améliorations du quatrième trimestre 2023

Cette page décrit toutes les autres améliorations apportées à l’environnement de prévisualisation par la version du quatrième trimestre 2023. Ces améliorations ont été apportées à l’environnement de production avec la version 23.10.

Pour une liste de tous les changements disponibles à ce stade du cycle de publication de la version du quatrième trimestre 2023, voir [Vue d’ensemble de la version du quatrième trimestre 2023](/help/quicksilver/product-announcements/product-releases/23-q4-release-activity/23-q4-release-overview.md).

## Modifications du menu principal du contributeur ou de la contributrice

Afin de mieux informer les personnes contributrices et demandeuses à propos des fonctionnalités disponibles avec un type de licence Workfront payant, elles peuvent désormais voir toutes les options disponibles dans le menu principal. Les options non disponibles pour les contributeurs et contributrices sont grisées.

Pour plus d’informations sur le menu principal, consultez [Vue d’ensemble de la barre de navigation supérieure](/help/quicksilver/workfront-basics/the-new-workfront-experience/global-navigation-overview.md).

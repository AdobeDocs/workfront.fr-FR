# Vue d’ensemble des intégrations intégrées d’Adobe Workfront

<!-- Audited: 12/2023 -->

Vous pouvez intégrer [!DNL Adobe Workfront] à des applications tierces, mais aussi à d’autres produits [!DNL Adobe]. Ces intégrations permettent d’étendre l’utilité de [!DNL Workfront] et de l’adapter aux besoins de votre organisation. Par exemple, si vous créez du contenu créatif dans le secteur marketing, vous pouvez utiliser le module complémentaire [!DNL Adobe Workfront] pour les applications [!DNL Creative Cloud] afin d’exporter directement des ressources de Photoshop vers Workfront.

Votre administrateur ou administratrice [!DNL Workfront] doit activer et configurer les intégrations avant que vous puissiez les utiliser. Pour savoir quelles intégrations sont utilisées au sein de votre organisation, contactez votre administrateur ou administratrice [!DNL Workfront].

## Autres intégrations [!DNL Workfront]

L’intégration souhaitée n’est peut-être pas une intégration native. Cependant, [!DNL Workfront] propose d’autres méthodes d’intégration pour l’application ou le service web de votre choix. Pour plus d’informations sur les autres méthodes d’intégration à [!DNL Workfront], consultez [Méthodes d’intégration à Adobe Workfront](../workfront-integrations-and-apps/built-in-vs-api-vs-fusion.md).

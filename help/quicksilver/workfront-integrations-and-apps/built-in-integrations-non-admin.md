# Vue d’ensemble des intégrations intégrées d’Adobe Workfront

<!-- Audited: 12/2023 -->

Vous pouvez intégrer [!DNL Adobe Workfront] à des applications tierces, ainsi qu’à d’autres produits [!DNL Adobe]. Ces intégrations peuvent étendre l’utilitaire de [!DNL Workfront] et l’adapter aux besoins de votre entreprise. Par exemple, si vous êtes un créatif dans le secteur marketing, vous pouvez utiliser le module externe [!DNL Adobe Workfront] pour les applications [!DNL Creative Cloud] afin d’exporter directement des ressources de Photoshop vers Workfront.

Votre administrateur [!DNL Workfront] doit activer et configurer les intégrations intégrées avant de pouvoir les utiliser. Pour savoir quelles intégrations intégrées sont utilisées au sein de votre organisation, contactez votre administrateur [!DNL Workfront].

## Autres [!DNL Workfront] intégrations

Pour plus d&#39;informations sur les autres méthodes d&#39;intégration à [!DNL Workfront], voir [Méthodes d&#39;intégration Adobe Workfront](../workfront-integrations-and-apps/built-in-vs-api-vs-fusion.md).

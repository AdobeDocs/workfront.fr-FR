# Installer [!DNL Adobe Workfront] pour les Microsofts Teams

<!-- Audited: 1/2024 -->

>[!IMPORTANT]
>
>L’intégration d’Adobe Workfront for Microsoft Teams n’est actuellement prise en charge que pour l’expérience de Microsofts Teams classiques.


L’application [!DNL Adobe Workfront for Microsoft Teams] vous permet d’effectuer des actions de base dans [!DNL Workfront] sans quitter vos canaux de conversation [!DNL Microsoft Teams].

>[!NOTE]
>
>[!DNL Microsoft Teams] ne prend plus en charge [!DNL Internet Explorer]. Pour utiliser le [!DNL Adobe Workfront for Microsoft Teams integration], vous devez utiliser un navigateur web autre que [!DNL Internet Explorer].

## Conditions préalables

Vous devez être propriétaire d’une équipe dans [!DNL Microsoft Teams] pour installer [!DNL Workfront] pour [!DNL Microsoft Teams].

## Installer [!DNL Workfront for Microsoft Teams]

En tant que propriétaire d’équipe dans [!DNL Microsoft Teams], vous pouvez installer l’application [!DNL Workfront for Microsoft Teams] pour chacune de vos équipes à partir de la boutique [!DNL Microsoft] ou d’un fichier fourni par [!DNL Workfront].

### Installez [!DNL Workfront for Microsoft Teams] à partir du [!DNL Microsoft] Store

1. Connectez-vous à [!DNL Microsoft Teams] en tant que propriétaire d’équipe.
1. Sélectionnez l’équipe pour laquelle vous souhaitez installer l’application [!DNL Workfront for Microsoft Teams].
1. Cliquez sur **[!UICONTROL Magasin]** dans la barre de navigation latérale.

1. Dans la zone **[!UICONTROL Rechercher tout]**, saisissez *[!DNL Workfront]*.

1. Cliquez sur la carte **[!DNL Workfront]** et suivez les instructions de l’assistant.
1. (Recommandé) Sélectionnez une équipe dans le menu déroulant **[!UICONTROL Ajouter à une équipe]** et activez l’option **[!UICONTROL Oui]** pour ajouter l’application à une équipe.

   ![ms_équipes_add_to_a_Team_option.png](assets/ms-teams-add-to-a-team-option-350x122.png)

1. Pour le canal, sélectionnez **[!UICONTROL Général]** pour utiliser l’application dans ce canal pour l’équipe sélectionnée, puis cliquez sur **[!UICONTROL Configurer]** pour les fonctionnalités de votre choix.

1. Une fois l’installation terminée, une notification indiquant que l’installation a réussi s’affiche dans le canal [!UICONTROL Général] de l’équipe que vous avez sélectionnée. Tous les membres de l’équipe peuvent voir cette notification.
1. (Facultatif) Épinglez votre application [!DNL Workfront] pour un accès plus facile :

   1. Cliquez sur l&#39;icône **[!UICONTROL Plus]** sous le champ de message dans le canal [!UICONTROL Général].

   1. Pointez sur l’application [!DNL Workfront] de la liste, puis cliquez sur l’icône **[!UICONTROL Plus]** située à droite de celle-ci.

   1. Cliquez sur **[!UICONTROL Pin]**.

      Une icône [!DNL Workfront] est alors ajoutée sous le champ de conversation. Vous pouvez accéder rapidement à la zone de [!UICONTROL recherche] à partir de cet emplacement.

      Pour plus d’informations sur la recherche d’éléments [!DNL Workfront], voir [Recherche et partage d’éléments [!DNL Adobe Workfront] dans [!DNL Microsoft Teams]](../../workfront-integrations-and-apps/using-workfront-with-microsoft-teams/search-for-and-share-wf-items-in-ms-teams.md).

1. Cliquez sur **[!UICONTROL Connectez-vous à[!DNL Workfront]]** pour accéder à [!DNL Workfront from Microsoft Teams].

   Pour plus d’informations sur la connexion à [!DNL Workfront], reportez-vous à la section [Connexion à Workfront à partir de Microsofts Teams](#log-in-to-workfront-from-microsoft-teams) de cet article.

### Installer [!DNL Workfront for Microsoft Teams] depuis un fichier privé

Si votre entreprise restreint l’accès au téléchargement des applications à partir de la boutique [!DNL Microsoft], vous devez contacter notre équipe d’assistance et demander un fichier privé de l’application [!DNL Workfront for Microsoft Teams] pour installer l’application.

Pour plus d’informations sur le contact avec notre équipe d’assistance, voir [Contacter le service clientèle](../../workfront-basics/tips-tricks-and-troubleshooting/contact-customer-support.md).

Pour installer [!DNL Workfront for Microsoft Teams] depuis un fichier privé :

1. Enregistrez le fichier privé que vous avez reçu de [!DNL Workfront] sur votre ordinateur.
1. Connectez-vous à [!DNL Microsoft Teams] en tant que propriétaire d’équipe [!DNL Microsoft].
1. Cliquez sur l&#39;icône **[!UICONTROL Plus]** pour l&#39;équipe pour laquelle vous souhaitez installer [!DNL Workfront for Microsoft Teams].

1. Cliquez sur **[!UICONTROL Gérer l’équipe]**.
1. Sélectionnez l’onglet **[!UICONTROL Applications]** , puis cliquez sur **[!UICONTROL Télécharger une application personnalisée]** dans le coin inférieur droit de l’écran.

1. Recherchez le fichier privé que vous avez enregistré sur votre ordinateur, puis suivez les étapes d’installation pour installer [!DNL Workfront for Microsoft Teams].
1. Une fois l’installation terminée, une notification indiquant que l’installation a réussi s’affiche dans le canal Général de l’équipe que vous avez sélectionnée. Tous les membres de l’équipe peuvent voir cette notification.
1. (Facultatif) Cliquez sur l’icône **[!UICONTROL Plus]** (trois points) sous le champ **[!UICONTROL Tapez vos questions ici]** .

1. (Facultatif) Pointez sur l’application [!DNL Workfront] de la liste, puis cliquez sur l’icône **[!UICONTROL Plus]** située à droite de celle-ci.

1. (Facultatif) Cliquez sur **[!UICONTROL Pin]**.

   Ceci ajoute une icône [!DNL Workfront] sous le champ [!UICONTROL Tapez vos questions ici] . Vous pouvez accéder rapidement à la zone de [!UICONTROL recherche] à partir de cet emplacement.\
   Pour plus d’informations sur la recherche d’éléments Workfront, voir [Recherche et partage [!DNL Adobe Workfront] d’éléments dans [!DNL Microsoft Teams]](../../workfront-integrations-and-apps/using-workfront-with-microsoft-teams/search-for-and-share-wf-items-in-ms-teams.md).

## Connectez-vous à [!DNL Workfront] à partir des [!DNL Microsoft] équipes

En tant que propriétaire d’équipe [!DNL Microsoft Teams], vous devez installer l’application [!DNL Workfront for Microsoft Teams] pour votre équipe avant que vous ou toute personne de l’équipe ne puisse vous connecter à [!DNL Workfront from Microsoft Teams].

Lorsque vous êtes connecté à [!DNL Workfront] à partir de [!DNL Microsoft Teams], vous pouvez recevoir [!DNL Workfront] notifications dans le canal de robot [!DNL Workfront] ou vous pouvez effectuer certaines actions dans [!DNL Workfront] à partir de [!DNL Microsoft Teams].

Pour plus d’informations sur l’installation de l’application [!DNL Workfront], consultez la section [Installation [!DNL Workfront for Microsoft Teams]](#install-workfront-for-microsoft-teams) de cet article.

Pour plus d’informations sur l’accès à [!DNL Workfront] à partir de [!DNL Microsoft Teams] pour effectuer certaines actions, voir [Accès [!DNL Adobe Workfront] à partir de [!DNL Microsoft Teams]](../../workfront-integrations-and-apps/using-workfront-with-microsoft-teams/access-workfront-from-ms-teams.md).

Pour vous connecter à [!DNL Workfront] à partir de [!DNL Microsoft Teams] :

1. Accédez au canal **[!UICONTROL Général]** de l’équipe où l’application [!DNL Workfront for Microsoft Teams] a été installée et cliquez sur **[!UICONTROL Connectez-vous à Workfront]**.

   Le canal de conversation de robot [!DNL Workfront] est ajouté à vos canaux de discussion [!DNL Microsoft Teams].

1. Accédez au canal de conversation de robot [!DNL Workfront] dans [!DNL Microsoft Teams] et saisissez *[!UICONTROL connectez-vous]* dans le champ **[!UICONTROL Saisissez vos questions ici]**.

   Ou

   Cliquez sur **[!UICONTROL Se connecter]**.

   Un nouvel onglet du navigateur s’ouvre.

1. Suivez les invites pour vous connecter à [!DNL Workfront] à l’aide de l’authentification améliorée, d’OAuth 2.0 ou de l’URL SAML (Security Assertion Markup Language).

   >[!NOTE]
   >
   >* Lorsque vous êtes invité à saisir le domaine de votre compte [!DNL Workfront], saisissez-le au format suivant : *yourCompany&#39;sDomain.my.workfront.com*. Le domaine de votre société correspond généralement au nom de votre société.
   >* L’authentification améliorée n’est pas disponible tant qu’un administrateur [!DNL Workfront] ne l’a pas activée pour cette intégration.


1. Fermez l’onglet du navigateur que vous avez utilisé pour vous connecter et revenez à [!DNL Microsoft Teams].

   Une notification s’affiche dans le canal de conversation de robot [!DNL Workfront] pour confirmer que vous vous êtes connecté à [!DNL Workfront] avec succès.

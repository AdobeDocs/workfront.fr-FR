# Installer [!DNL Adobe Workfront] Microsofts Teams

<!-- Audited: 1/2024 -->

>[!IMPORTANT]
>
>L’intégration d’Adobe Workfront for Microsoft Teams n’est actuellement prise en charge que pour l’expérience de Microsofts Teams classiques.


La variable [!DNL Adobe Workfront for Microsoft Teams] vous permet d’effectuer des actions de base dans [!DNL Workfront] sans quitter le [!DNL Microsoft Teams] canaux de discussion.

>[!NOTE]
>
>[!DNL Microsoft Teams] ne prend plus en charge [!DNL Internet Explorer]. Pour utiliser la variable [!DNL Adobe Workfront for Microsoft Teams integration], vous devez utiliser un navigateur web autre que [!DNL Internet Explorer].

## Conditions préalables

Vous devez être propriétaire d’une équipe dans [!DNL Microsoft Teams] pour installer [!DNL Workfront] pour [!DNL Microsoft Teams].

## Installer [!DNL Workfront for Microsoft Teams]

En tant que propriétaire d’équipe dans [!DNL Microsoft Teams], vous pouvez installer le [!DNL Workfront for Microsoft Teams] pour chacune de vos équipes, soit à partir du [!DNL Microsoft] Stocker ou dans un fichier fourni par [!DNL Workfront].

### Installer [!DNL Workfront for Microsoft Teams] de la [!DNL Microsoft] Magasin

1. Connexion à [!DNL Microsoft Teams] en tant que propriétaire d’équipe.
1. Sélectionnez l’équipe pour laquelle vous souhaitez installer le [!DNL Workfront for Microsoft Teams] application.
1. Cliquez sur **[!UICONTROL Magasin]** dans la barre de navigation latérale.

1. Dans le **[!UICONTROL Rechercher tout]** box, type *[!DNL Workfront]*.

1. Cliquez sur le bouton **[!DNL Workfront]** et suivez les instructions de l’assistant.
1. (Recommandé) Sélectionnez une équipe dans le **[!UICONTROL Ajout à une équipe]** et activez l’option **[!UICONTROL Oui]** pour ajouter l’application à une équipe.

   ![ms_équipes_add_to_a_Team_option.png](assets/ms-teams-add-to-a-team-option-350x122.png)

1. Pour le canal, sélectionnez **[!UICONTROL Général]** pour utiliser l’application dans ce canal pour l’équipe sélectionnée, puis cliquez sur **[!UICONTROL Configuration]** pour les fonctionnalités de votre choix.

1. Une fois l’installation terminée, une notification indiquant que l’installation a réussi s’affiche dans la variable [!UICONTROL Général] canal de l’équipe que vous avez sélectionnée. Tous les membres de l’équipe peuvent voir cette notification.
1. (Facultatif) Epinglez votre [!DNL Workfront] pour un accès plus facile :

   1. Cliquez sur le bouton **[!UICONTROL Plus]** sous le champ message de la [!UICONTROL Général] canal.

   1. Passez la souris sur la [!DNL Workfront] dans la liste, puis cliquez sur le bouton **[!UICONTROL Plus]** à droite de celle-ci.

   1. Cliquez sur **[!UICONTROL Epinglage]**.

      Cela ajoute une [!DNL Workfront] dans le champ de conversation. Vous pouvez accéder rapidement au [!UICONTROL Rechercher] d’ici.

      Pour plus d’informations sur la recherche de [!DNL Workfront] éléments, voir [Rechercher et partager [!DNL Adobe Workfront] éléments dans [!DNL Microsoft Teams]](../../workfront-integrations-and-apps/using-workfront-with-microsoft-teams/search-for-and-share-wf-items-in-ms-teams.md).

1. Cliquez sur **[!UICONTROL Connexion à[!DNL Workfront]]** accès [!DNL Workfront from Microsoft Teams].

   Pour plus d’informations sur la connexion à [!DNL Workfront], voir [Connexion à Workfront à partir de Microsofts Teams](#log-in-to-workfront-from-microsoft-teams) dans cet article.

### Installer [!DNL Workfront for Microsoft Teams] à partir d’un fichier privé

Si votre entreprise restreint l’accès au téléchargement d’applications à partir de la variable [!DNL Microsoft] Magasin, vous devez contacter notre équipe d’assistance et demander un fichier privé du [!DNL Workfront for Microsoft Teams] pour installer l’application.

Pour plus d’informations sur le contact avec notre équipe d’assistance, voir [Contacter le service clientèle](../../workfront-basics/tips-tricks-and-troubleshooting/contact-customer-support.md).

Pour installer [!DNL Workfront for Microsoft Teams] à partir d’un fichier privé :

1. Enregistrez le fichier privé que vous avez reçu [!DNL Workfront] sur votre ordinateur.
1. Connexion à [!DNL Microsoft Teams] as a [!DNL Microsoft] propriétaire de l’équipe.
1. Cliquez sur le bouton **[!UICONTROL Plus]** pour l’équipe pour laquelle vous souhaitez installer [!DNL Workfront for Microsoft Teams].

1. Cliquez sur **[!UICONTROL Gérer l’équipe]**.
1. Sélectionnez la variable **[!UICONTROL Applications]** , puis cliquez sur **[!UICONTROL Téléchargement d’une application personnalisée]** dans le coin inférieur droit de l’écran.

1. Recherchez le fichier privé que vous avez enregistré sur votre ordinateur, puis suivez les étapes d’installation pour l’installer. [!DNL Workfront for Microsoft Teams].
1. Une fois l’installation terminée, une notification indiquant que l’installation a réussi s’affiche dans le canal Général de l’équipe que vous avez sélectionnée. Tous les membres de l’équipe peuvent voir cette notification.
1. (Facultatif) Cliquez sur le **[!UICONTROL Plus]** (trois points) sous l’icône **[!UICONTROL Saisissez vos questions ici]** champ .

1. (Facultatif) Placez le pointeur de la souris sur la variable [!DNL Workfront] dans la liste, puis cliquez sur le bouton **[!UICONTROL Plus]** à droite de celle-ci.

1. (Facultatif) Cliquez sur **[!UICONTROL Epinglage]**.

   Cela ajoute une [!DNL Workfront] sous l’icône [!UICONTROL Saisissez vos questions ici] champ . Vous pouvez accéder rapidement au [!UICONTROL Rechercher] d’ici.\
   Pour plus d’informations sur la recherche d’éléments Workfront, voir [Rechercher et partager [!DNL Adobe Workfront] éléments dans [!DNL Microsoft Teams]](../../workfront-integrations-and-apps/using-workfront-with-microsoft-teams/search-for-and-share-wf-items-in-ms-teams.md).

## Connexion à [!DNL Workfront] de [!DNL Microsoft] Équipes

Comme [!DNL Microsoft Teams] propriétaire de l’équipe, vous devez installer [!DNL Workfront for Microsoft Teams] avant que vous ou toute personne de l’équipe ne puissiez vous connecter à [!DNL Workfront from Microsoft Teams].

Lorsque vous êtes connecté à [!DNL Workfront] de [!DNL Microsoft Teams], vous pouvez recevoir [!DNL Workfront] notifications dans la variable [!DNL Workfront] ou vous pouvez effectuer certaines actions dans [!DNL Workfront] de [!DNL Microsoft Teams].

Pour plus d’informations sur l’installation du [!DNL Workfront] , voir [Installer [!DNL Workfront for Microsoft Teams]](#install-workfront-for-microsoft-teams) dans cet article.

Pour plus d’informations sur l’accès à [!DNL Workfront] de [!DNL Microsoft Teams] pour effectuer certaines actions, voir [Accès [!DNL Adobe Workfront] de [!DNL Microsoft Teams]](../../workfront-integrations-and-apps/using-workfront-with-microsoft-teams/access-workfront-from-ms-teams.md).

Pour vous connecter à [!DNL Workfront] de [!DNL Microsoft Teams]:

1. Accédez au **[!UICONTROL Général]** canal de l’équipe dans laquelle la fonction [!DNL Workfront for Microsoft Teams] L’application a été installée et cliquez sur **[!UICONTROL Connexion à Workfront]**.

   La variable [!DNL Workfront] Ajout du canal de chat de robots à votre [!DNL Microsoft Teams] canaux de discussion.

1. Accédez au [!DNL Workfront] Chaîne de conversation de robot dans [!DNL Microsoft Teams] et type *[!UICONTROL connexion]* dans le **[!UICONTROL Saisissez vos questions ici]** champ .

   Ou

   Cliquez sur **[!UICONTROL Connexion]**.

   Un nouvel onglet du navigateur s’ouvre.

1. Suivez les invites pour vous connecter à [!DNL Workfront] à l’aide d’une authentification améliorée, d’OAuth 2.0 ou de l’URL SAML (Security Assertion Markup Language).

   >[!NOTE]
   >
   >* Lorsque vous êtes invité à saisir le domaine de votre [!DNL Workfront] compte, saisissez-le au format suivant : *yourCompany&#39;sDomain.my.workfront.com*. Le domaine de votre société correspond généralement au nom de votre société.
   >* L’authentification améliorée n’est pas disponible tant qu’un [!DNL Workfront] L’administrateur l’active pour cette intégration.


1. Fermez l’onglet du navigateur dans lequel vous vous êtes connecté et revenez à [!DNL Microsoft Teams].

   Une notification s’affiche dans la variable [!DNL Workfront] Chaîne de conversation de robot pour confirmer que vous êtes connecté à [!DNL Workfront] réussi.

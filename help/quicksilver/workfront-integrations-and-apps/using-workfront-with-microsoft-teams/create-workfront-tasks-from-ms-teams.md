## Conditions préalables

Vous pouvez créer des tâches personnelles dans [!DNL Adobe Workfront] à partir de [!DNL Microsoft Teams] si les conditions suivantes sont remplies :

* Un propriétaire d’équipe a installé et configuré [!DNL Workfront for Microsoft Teams] pour votre équipe.
* Vous êtes connecté à [!DNL Workfront] à partir de [!DNL Microsoft Teams].

>[!NOTE]
>
>[!DNL Microsoft Teams] ne prend plus en charge [!DNL Internet Explorer]. Pour utiliser l&#39;intégration [!DNL Adobe Workfront for Microsoft Teams], vous devez utiliser un navigateur web autre que [!DNL Internet Explorer].

Pour plus d&#39;informations sur l&#39;installation de [!DNL Workfront for Microsoft Teams] et la connexion à [!UICONTROL Workfront] à partir de [!DNL Microsoft Teams], voir [Installation [!DNL Adobe Workfront for Microsoft Teams]](../../workfront-integrations-and-apps/using-workfront-with-microsoft-teams/install-workfront-ms-teams.md).

## Créer des tâches personnelles à partir de [!DNL Microsoft Teams]

1. Connectez-vous à [!DNL Workfront] depuis [!DNL Microsoft Teams].

   Pour plus d&#39;informations sur la connexion à [!DNL Workfront], voir [Installation [!DNL Adobe Workfront for Microsoft Teams]](../../workfront-integrations-and-apps/using-workfront-with-microsoft-teams/install-workfront-ms-teams.md).

1. Pour ouvrir une carte **[!UICONTROL Nouvelle tâche]** :

   * Si vous vous trouvez dans le canal de conversation de robot [!DNL Workfront], saisissez **[!UICONTROL Nouvelle tâche]** dans le champ [!UICONTROL conversation] pour créer une nouvelle tâche.
   * Si vous utilisez un canal de conversation autre que le canal de discussion de bots [!DNL Workfront] :

      * Commencez à taper **[!DNL @workfront]** dans le champ [!UICONTROL conversation] , puis sélectionnez le canal de robot [!DNL Workfront] de votre choix.
      * Continuez à taper **[!UICONTROL Nouvelle tâche]** dans le champ [!UICONTROL conversation] pour créer une nouvelle tâche.

        La carte [!UICONTROL Nouvelle tâche] s’affiche dans le canal de robot [!DNL Workfront].

        ![ms_équipes_new_task_card.png](assets/ms-teams-new-task-card-350x181.png)

1. Dans le canal de robot [!UICONTROL Workfront], spécifiez les informations suivantes sur la carte [!UICONTROL Nouvelle tâche] :

   * Nom de la tâche dans le champ **[!UICONTROL Ecrire le titre]** de la tâche.
   * Description de la tâche dans le champ **[!UICONTROL Ecrire la description de la tâche]**.
   * Date à laquelle la tâche doit être terminée, dans le champ **[!UICONTROL Date d’échéance]**.

1. Cliquez sur **[!UICONTROL Enregistrer].**

   La nouvelle tâche personnelle est créée dans [!DNL Workfront]. Un [!UICONTROL numéro de référence] lui est attribué et est visible sur la carte [!UICONTROL nouvelle tâche].

   Pour plus d’informations sur les numéros de référence, reportez-vous à la section [[!UICONTROL Numéro de référence] des objets](../../workfront-basics/navigate-workfront/workfront-navigation/understand-objects.md#understanding-reference-numbers-of-objects) de l’article [Comprendre les objets dans [!DNL Adobe Workfront]](../../workfront-basics/navigate-workfront/workfront-navigation/understand-objects.md) .

1. (Facultatif) Cliquez sur **[!UICONTROL Modifier]** pour modifier les informations de la tâche.
1. (Facultatif) Cliquez sur **[!UICONTROL Afficher dans[!DNL Workfront]]** pour ouvrir la tâche dans un nouvel onglet de [!DNL Workfront] et modifier la tâche, la déplacer vers un projet ou l’affecter à une autre personne.

# Gérer votre instance d’Adobe Workfront Planning en tant que produit autonome

>[!IMPORTANT]
>
>Les informations de cet article font référence à Adobe Workfront Planning, lorsqu’il est acheté en tant que produit autonome. Reportez-vous à cet article lorsque votre société a acheté un package Adobe Workfront Planning uniquement sans acheter de package Workfront Workflow.
>
>Pour plus d’informations sur Adobe Workfront Planning lorsqu’il est acheté avec un package Workfront, voir [Prise en main d’Adobe Workfront Planning](/help/quicksilver/planning/general/planning-overview.md).

En tant qu’administrateur ou administratrice Workfront Planning, vous pouvez gérer votre instance Adobe Workfront Planning.

Vous pouvez configurer les paramètres suivants pour Workfront Planning :

* Trimestres personnalisés
* Infos client
* Préférences
* Équipes

  Pour plus d’informations, voir [Gérer les équipes dans Adobe Workfront Planning en tant que produit autonome](/help/quicksilver/planning/planning-sta/manage-teams-in-planning-sta.md).
* Utilisateurs et utilisatrices

  Pour plus d’informations, voir [Gérer les utilisateurs dans Adobe Workfront Planning en tant que produit autonome](/help/quicksilver/planning/planning-sta/manage-users-in-planning-sta.md).

La configuration de Workfront Planning en tant que produit autonome est similaire à la configuration de certains paramètres dans Adobe Workfront. Cependant, il existe des limitations quant aux informations disponibles dans Workfront Planning.

## Se connecter sous une autre identité

En tant qu&#39;administrateur Planning, vous pouvez vous connecter en tant qu&#39;utilisateur différent afin de résoudre les problèmes qui se posent à lui. La connexion en tant qu’autre utilisateur dans Workfront Planning est similaire à la connexion en tant qu’autre utilisateur dans Workfront.

Pour plus d’informations, consultez la section [Se connecter en tant qu’un autre utilisateur ou qu’une autre utilisatrice](/help/quicksilver/administration-and-setup/add-users/create-and-manage-users/log-in-as-another-user.md).

## Configuration des trimestres personnalisés

Vous pouvez créer des trimestres personnalisés si les trimestres de votre entreprise sont basés sur des critères spécifiques autres que des dates calendaires (par exemple, des jours ouvrables ou des jours ouvrables).

Vous pouvez configurer jusqu’à 100 trimestres pour votre instance Workfront Planning. Vous pouvez utiliser ces trimestres personnalisés dans la vue chronologique.

La configuration des trimestres personnalisés dans Workfront Planning est similaire à la configuration des trimestres personnalisés dans Workfront.

Pour plus d’informations, voir [Activer les trimestres personnalisés](/help/quicksilver/administration-and-setup/set-up-workfront/configure-system-defaults/enable-custom-quarters-projects.md).

## Configurer les informations sur le client

Dans le cadre de la configuration de votre produit Adobe Workfront Planning, vous pouvez gérer les détails de votre organisation dans la section Informations sur le client de la configuration.

La configuration de la section Informations du client dans Workfront Planning est similaire à sa configuration dans Workfront.

Pour plus d’informations, consultez la section [Configurer des informations de base pour votre système](/help/quicksilver/administration-and-setup/get-started-wf-administration/configure-basic-info.md).


## Configuration des préférences du système

En tant qu&#39;administrateur de Planning, vous pouvez configurer des préférences pour votre système de planification Workfront.

Les modifications apportées à la zone Préférences système de la Configuration ont un impact sur tous les utilisateurs de votre système et sur leur expérience dans Workfront Planning.

Nous vous recommandons de configurer vos préférences système au cours de la planification Workfront en tant qu’implémentation de produit autonome et de ne les revoir qu’occasionnellement par la suite.

La configuration des préférences système de Workfront Planning est similaire à leur configuration dans Workfront.

Pour plus d’informations, voir [ Configuration des préférences système ](/help/quicksilver/administration-and-setup/manage-workfront/security/configure-security-preferences.md).

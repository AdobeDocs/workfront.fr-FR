# Connexion aux types d’enregistrement - Aperçu

Une façon d’indiquer comment les types d’enregistrements individuels sont associés les uns aux autres consiste à les connecter. En outre, vous pouvez connecter les types d’enregistrements Adobe Workfront Planning aux types d’objets d’autres applications afin d’améliorer l’expérience de vos utilisateurs et utilisatrices et de garder leur attention sur une seule application.

Cet article présente la manière dont les types d’enregistrement se connectent et décrit les types de connexions que vous pouvez établir entre les types d’enregistrement et d’objet.

# Créer, modifier et afficher les programmes d’un groupe

Lorsque vous affichez un groupe que vous gérez dans la zone Groupes , vous pouvez afficher et utiliser ses programmes.

S’il existe des groupes au-dessus du groupe que vous gérez, leur équipe d’administration peut également le faire pour votre groupe. Il en va de même pour l’administration de Workfront (pour n’importe quel groupe).

## Affichage, utilisation et création de programmes pour votre groupe à partir de la zone Groupes

{{step-1-to-setup}}

1. Dans le panneau de gauche, cliquez sur **Groupes** ![](assets/groups-icon.png).

1. Cliquez sur le nom du groupe pour lequel vous souhaitez créer ou modifier des programmes.
1. Dans le panneau de gauche, cliquez sur **Programmes** pour répertorier les programmes associés au groupe auquel vous avez accès (ainsi que tous les sous-groupes situés en dessous).
1. Effectuez l’une des opérations suivantes :

   <table style="table-layout:auto"> 
    <col> 
    <col> 
    <tbody> 
     <tr> 
      <td role="rowheader">Ajout d’un programme</td> 
      <td> 
       <ol> 
        <li value="1"> <p>Cliquez sur <strong>Nouveau programme</strong>, puis configurez le programme dans l’écran qui s’affiche. </p> <p>Le système renseigne le champ <strong>Group</strong> pour le nouveau programme avec le groupe que vous affichez.</p> <p>Pour plus d'informations sur les options de configuration, voir <a href="../../../manage-work/portfolios/create-and-manage-programs/create-program.md" class="MCXref xref">Création d'un programme</a>.</p> </li> 
        <li value="2"> <p>Cliquez sur <strong>Enregistrer</strong> pour revenir à la liste Programmes du groupe.</p> </li> 
       </ol> </td> 
     </tr> 
     <tr> 
      <td role="rowheader"> <p>Modification ou suppression de programmes</p> </td> 
      <td> <p>Sélectionnez au moins un programme que vous êtes autorisé à modifier, puis utilisez les boutons de la barre d'outils pour le modifier <img src="assets/edit-icon.png"> ou le supprimer <img src="assets/delete.png">.</p> <p>Pour plus d'informations sur la modification d'un programme, voir <a href="../../../manage-work/portfolios/create-and-manage-programs/create-program.md" class="MCXref xref">Création d'un programme</a>.</p> </td> 
     </tr>

     </tr> 
     <tr> 
      <td role="rowheader"> <p>Exporter la liste des programmes</p> </td> 
      <td>Cliquez sur <strong>Export</strong> <img src="assets/export.png">, puis sélectionnez le format de fichier souhaité pour la liste exportée.</td>

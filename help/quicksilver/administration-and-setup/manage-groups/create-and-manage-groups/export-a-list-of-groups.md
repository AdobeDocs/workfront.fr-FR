# Exporter une liste de groupes

Lorsque vous affichez les groupes que vous gérez, vous pouvez exporter une liste de ces groupes.

S’il existe des groupes au-dessus du groupe que vous gérez, leurs administrateurs et administratrices peuvent également le faire pour votre groupe. Il en va de même pour les administrateurs et administratrices de Workfront (pour n’importe quel groupe).

## Exporter les groupes

{{step-1-to-setup}}

1. Dans le panneau de gauche, cliquez sur **Groupes** ![Groupes](assets/groups-icon.png).

   Dans la liste qui s’affiche, vous pouvez voir les groupes que vous gérez, ainsi que les sous-groupes qu’ils comportent. Les administrateurs et administratrices Adobe Workfront peuvent voir tous les groupes.

1. Si vous souhaitez exporter l’ensemble du groupe de groupes géré, cliquez sur l’icône Exporter ![icône Exporter](assets/export.png).

   Ou

   Pour exporter certains des groupes gérés, sélectionnez-les, puis cliquez sur l’icône Exporter ![icône Exporter](assets/export.png).

1. Sélectionnez le format de fichier souhaité de la liste exportée.

## Permettre aux utilisateurs d&#39;accéder à l&#39;application mobile

1. Cliquez sur le bouton **[!UICONTROL Menu Principal]** icon ![](assets/main-menu-icon.png) dans le coin supérieur droit de [!DNL Adobe Workfront], puis cliquez sur **[!UICONTROL Configuration]** ![](assets/gear-icon-settings.png).

1. Cliquez sur **[!UICONTROL Configuration],** situé dans le coin supérieur droit de la page.

1. Développer **[!UICONTROL Système],** puis cliquez sur **[!UICONTROL Préférences].**

1. Sélectionnez la **[!UICONTROL Permet aux utilisateurs d’utiliser Workfront pour les applications mobiles et la variable [!DNL Workfront Outlook] Module complémentaire]** pour permettre aux utilisateurs d’utiliser [!DNL Workfront] avec le [!DNL Workfront] application mobile ou pour installer et utiliser [!DNL Workfront] pour Outlook.\
   Cette option est activée par défaut.

1. Cliquer sur **[!UICONTROL Enregistrer].**

Pour plus d’informations sur l’utilisation de la variable [!DNL Workfront] une fois l’application mobile activée pour le site, voir [Utilisez la variable [!DNL Adobe Workfront] application mobile](../../workfront-basics/mobile-apps/using-the-workfront-mobile-app/use-the-mobile-app.md).

Pour plus d’informations sur l’utilisation de [!DNL Workfront for Outlook], reportez-vous à la section [Utilisation [!DNL Workfront with Outlook]](../../workfront-integrations-and-apps/using-workfront-with-outlook/workfront-for-outlook.md) .

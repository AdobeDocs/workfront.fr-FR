# Créer des rapports directs

Vous pouvez désigner des utilisateurs et utilisatrices comme subordonnés directs d’un autre utilisateur ou une autre utilisatrice. Cela vous permet de créer un organigramme qui affiche la structure de gestion de votre organisation. Pour plus d’informations, consultez [Afficher l’organigramme](../../../people-teams-and-groups/work-directly-with-others/view-the-org-chart.md).

## Créer des rapports directs

1. Commencez à modifier un utilisateur ou une utilisatrice comme décrit dans [Modifier le profil d’un utilisateur ou d’une utilisatrice](../../../administration-and-setup/add-users/create-and-manage-users/edit-a-users-profile.md).
1. Dans la section **Organisation**, assurez-vous qu’une entreprise est sélectionnée dans le champ **Entreprise**.

   Ce champ doit obligatoirement être renseigné.

1. Dans le champ **Rapports directs**, indiquez les utilisateurs et utilisatrices qui dépendent de l’utilisateur ou utilisatrice que vous êtes en train de modifier.
1. (Facultatif) Dans le champ **Rend compte à**, indiquez le nom de la personne à laquelle cet utilisateur ou cette utilisatrice rend compte.
1. Cliquez sur **Enregistrer les modifications**.

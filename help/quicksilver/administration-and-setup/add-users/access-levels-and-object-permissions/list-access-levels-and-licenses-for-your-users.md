## Répertorier les niveaux d’accès et les licences de vos utilisateurs

1. Cliquez sur le bouton **Menu Principal** icon ![](assets/main-menu-icon.png) dans le coin supérieur droit d’Adobe Workfront, puis cliquez sur **Utilisateurs**.

   Par défaut, tous les utilisateurs définis comme Principaux s’affichent dans la liste.

1. Dans le **Affichage** menu déroulant, sélectionnez **Licences**.

   La colonne Licence affiche le nom de la licence affectée à chaque utilisateur.

1. Dans le **Regroupement** menu déroulant, cliquez sur **Type de licence**.

   Cette vue regroupe les utilisateurs qui ont les mêmes types de licence.

1. (Facultatif) Pour filtrer la liste selon une licence spécifique :

   1. Cliquez sur le bouton **Filtrer** menu déroulant, puis cliquez sur **Nouveau filtre**.

   1. Cliquez sur **Ajouter une règle de filtre**.
   1. Commencez à taper **Licence** et sélectionnez-le lorsqu’il apparaît dans la liste.
   1. Avec le **Égal** sélectionnez cette option, commencez à saisir le nom de la licence selon laquelle vous souhaitez filtrer la liste.

      Vous pouvez spécifier plusieurs types de licence.

   1. Cliquez sur **Enregistrer le filtre**.

      La liste affiche uniquement les utilisateurs associés aux types de licence que vous avez spécifiés dans le filtre.
   >[!TIP]
   >
   >Vous pouvez également regrouper la liste par type de licence ou la filtrer selon une licence spécifique.

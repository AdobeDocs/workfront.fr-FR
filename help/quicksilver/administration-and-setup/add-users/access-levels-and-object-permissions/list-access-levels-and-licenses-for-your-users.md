## Liste des niveaux d’accès et des licences de vos utilisateurs

1. Cliquez sur l’icône **Menu principal** ![](assets/main-menu-icon.png) dans le coin supérieur droit d’Adobe Workfront, puis cliquez sur **Utilisateurs**.

   Par défaut, tous les utilisateurs définis comme actifs s’affichent dans la liste.

1. Dans le menu déroulant **Afficher**, sélectionnez **Licences**.

   La colonne Licence affiche le nom de la licence affectée à chaque utilisateur.

1. Dans le menu déroulant **Groupement**, cliquez sur **Type de licence**.

   Cette vue regroupe les utilisateurs qui ont les mêmes types de licence.

1. (Facultatif) Pour filtrer la liste selon une licence spécifique :

   1. Cliquez sur le menu déroulant **Filtre**, puis sur **Nouveau filtre**.

   1. Cliquez sur **Ajouter une règle de filtre**.
   1. Commencez à saisir **Licence** et sélectionnez-le lorsqu’il apparaît dans la liste.
   1. Avec le modificateur **Equal** sélectionné, commencez à saisir le nom de la licence dont vous souhaitez filtrer la liste par .

      Vous pouvez spécifier plusieurs types de licence.

   1. Cliquez sur **Enregistrer le filtre**.

      La liste affiche uniquement les utilisateurs associés aux types de licence que vous avez spécifiés dans le filtre.

   >[!TIP]
   >
   >Vous pouvez également regrouper la liste par type de licence ou la filtrer selon une licence spécifique.

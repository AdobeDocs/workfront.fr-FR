# [!DNL Adobe Workfront] assistant virtuel d&#39;application mobile

Utilisez les commandes vocales suivantes pour gérer le travail dans l’application mobile :

* [[!UICONTROL Supprimer]](#delete)
* [[!UICONTROL Aide-moi avec des commandes]](#help-me-with-commands)

## Commandes Pin

Utilisez la fonction d’épingle pour accéder aux commandes enregistrées de l’assistant virtuel, ainsi qu’aux broches de votre navigateur web. Une fois que vous avez épinglé une commande, vous pouvez y accéder à tout moment à partir de la fonction [!UICONTROL Epinglage] zone.

### Assistant virtuel (VA)

La section VA de la section [!UICONTROL Epinglage] vous permet d’enregistrer les commandes vocales pour y accéder ultérieurement.

Pour épingler une commande :

1. Saisissez votre commande, puis appuyez sur **[!UICONTROL Epinglage]** sur la page Résultats .

![Epingler et enregistrer la requête](assets/pin-and-save-query-adobe-350x285.png)

### Navigateur web

Le [!UICONTROL Web] de la section [!UICONTROL Epinglage] vous permet d’accéder aux objets que vous avez épinglés dans votre navigateur web. Si l’objet épinglé est pris en charge dans l’application mobile, il s’ouvre en premier. S’il n’est pas pris en charge, il lance un navigateur web.

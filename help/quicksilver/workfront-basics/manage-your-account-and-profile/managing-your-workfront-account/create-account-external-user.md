# Créer un compte dans [!DNL Adobe Workfront] en tant qu’utilisateur externe

Vous pouvez être invité à afficher ou approuver des éléments dans [!DNL Workfront] même si vous n&#39;êtes pas membre d&#39;une organisation [!DNL Workfront]. Par exemple, vous pouvez être invité à réviser ou approuver des documents. Vous pouvez afficher ou approuver ces éléments plus facilement en créant un compte dans [!DNL Workfront].

Lorsque l’utilisateur dans [!DNL Workfront] vous demande de réviser ou d’approuver un document, <!--or shares a Workfront object such as a report or Board with you, -->vous recevez un e-mail. Cliquez sur le lien contenu dans l’e-mail pour accéder directement à l’élément partagé avec vous.

La création d’un compte dans [!DNL Workfront] vous permet d’afficher tous les éléments partagés avec vous ou demandés à votre place. Sans compte, vous ne pouvez accéder à ces éléments que par le biais de liens d’e-mail individuels.

## Créer un compte

Vous pouvez créer un compte lors de l’affichage de toute ressource partagée avec vous.

1. Sur la page des ressources, cliquez sur **Terminer la configuration du compte** dans l’en-tête du document.

1. Renseignez la section **Mot de passe**.

   Pour plus d’informations, voir [ Mot de passe ](#password).

1. (Facultatif) Renseignez la section **Profil**.

   Pour plus d’informations, voir [Profil](#profile).

1. Cliquez sur **Créer un compte**.


Lorsque la page Créer un compte s’affiche, renseignez les informations suivantes.

### Mot de passe

>[!NOTE]
>
>Le nom d’utilisateur de votre compte Workfront est votre adresse e-mail.

Créer un nouveau mot de passe pour votre compte

Les mots de passe doivent comporter au moins 8 caractères et au moins deux des éléments suivants :

* Lettres majuscules
* Lettres minuscules
* Nombres
* Caractères spéciaux

### Profil

La création d&#39;un profil est facultative, mais recommandée. Votre nom et votre image permettent à d’autres utilisateurs de vérifier plus facilement, par exemple, que c’est vous qui avez affecté l’approbation d’un document donné ou qui vous trouvez dans une liste lors de l’affectation de réviseurs.

## Connexion à Workfront

Après avoir créé votre compte , vous pouvez vous connecter à Workfront pour afficher les éléments partagés avec vous ou qui vous sont affectés.

* Le nom d’utilisateur de votre compte Workfront est votre adresse e-mail.
* Le domaine de votre compte Workfront est le domaine de l’organisation qui vous a envoyé la demande.

  Vous pouvez trouver le domaine de l’organisation en cliquant sur le lien contenu dans l’e-mail, puis en vérifiant l’URL du document. Le domaine s’affiche au début de l’URL.

  `https://<Domain is here>.my.workfront.com/...`

Pour obtenir des instructions, voir [Connexion à Workfront](/help/quicksilver/workfront-basics/manage-your-account-and-profile/managing-your-workfront-account/log-in-to-workfront.md).

## Utiliser les clés dans [!DNL Workfront Fusion]

Si un module nécessite une clé publique ou privée, vous pouvez l’ajouter au trousseau. Pour plus d’informations, voir [Configurer les paramètres d’un module dans  [!DNL Adobe Workfront Fusion]](../../workfront-fusion/modules/configure-a-modules-settings.md).

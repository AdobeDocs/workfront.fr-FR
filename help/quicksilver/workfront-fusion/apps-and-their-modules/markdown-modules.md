# [!UICONTROL Markdown] modules

Si vous avez besoin d’instructions sur la création d’un scénario, reportez-vous à la section [Création d’un scénario dans [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/create-a-scenario.md).

## [!UICONTROL Marquage au HTML]

Ce module convertit Markdown en HTML.

Lors de la configuration de ce module, les champs suivants s’affichent.

<table style="table-layout:auto"> 
 <col> 
 <col> 
 <tbody> 
  <tr> 
   <td role="rowheader">[!UICONTROL Markdown]</td> 
   <td> <p>Saisissez le texte brut au format Markdown.</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader">[!DNL GitHub Flavored Markdown] </td> 
   <td> <p>Activez cette option pour convertir GitHub Flavored Markdown en HTML.</p> <p>Pour plus d’informations, voir[!DNL ]Aide-mémoire dans la [!DNL GitHub] documentation.</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader">[!UICONTROL Sanitize]</td> 
   <td>Sélectionnez une option pour indiquer si vous souhaitez supprimer les balises de HTML du texte ou le HTML d’échappement.</td> 
  </tr> 
 </tbody> 
</table>

## [!UICONTROL HTML vers Markdown]

Ce module convertit le code de HTML en Markdown.

<table style="table-layout:auto"> 
 <col> 
 <col> 
 <tbody> 
  <tr> 
   <td role="rowheader">[!UICONTROL Markdown]</td> 
   <td> <p>Saisissez le code de HTML à convertir en Markdown.</p> </td> 
  </tr> 
  <tr> 
   <td role="rowheader">[!DNL GitHub Flavored Markdown] </td> 
   <td> <p>Activez cette option pour convertir le HTML en [!DNL GitHub Flavored Markdown].</p> <p>Pour plus d’informations, reportez-vous à la feuille de calcul Markdown de la section [!DNL GitHub] documentation.</p> </td> 
  </tr> 
 </tbody> 
</table>

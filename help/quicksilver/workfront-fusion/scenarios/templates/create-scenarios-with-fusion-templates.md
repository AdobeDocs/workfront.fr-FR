## Conditions préalables

Vous devez créer un modèle avant de pouvoir l’utiliser.

## Créer un scénario à partir d’un modèle

1. Cliquez sur l’icône **[!UICONTROL Modèles]** dans le panneau de navigation de gauche.![](assets/fusion-template-icon.png)
1. Cliquez sur l’onglet contenant le modèle que vous souhaitez utiliser (**[!UICONTROL Modèles publics]** ou **[!UICONTROL Modèles d’équipe]**).
1. Cliquez sur le modèle à utiliser pour créer un scénario. Vous pouvez utiliser la fonction de recherche pour trouver le modèle par son nom ou le nom de l’application qu’il contient.
1. (Conditionnel) Si vous êtes membre de plusieurs organisations ou équipes, sélectionnez l’organisation ou l’équipe pour laquelle vous souhaitez créer votre scénario, puis cliquez sur **[!UICONTROL Continuer]**.
1. (Conditionnel) Si le modèle a été configuré avec [!UICONTROL Wizard], suivez les invites pour configurer le scénario. Cliquez sur **[!UICONTROL Continuer]** à chaque invite pour le confirmer.

   >[!NOTE]
   >
   >Si l&#39;option **[!UICONTROL Utiliser comme valeur par défaut]** a été cochée lors de la configuration du modèle, certains champs peuvent être préremplis pour vous.

1. Lorsque vous avez configuré tous les paramètres requis pour le modèle, vous recevez une notification indiquant que le modèle est prêt à être utilisé. Les options du scénario standard s’affichent en bas de l’écran. Vous pouvez maintenant continuer à créer votre scénario.

   Pour plus d&#39;informations sur la création d&#39;un scénario, voir [Création d&#39;un scénario dans [!DNL Adobe Workfront Fusion]](../../../workfront-fusion/scenarios/create-a-scenario.md).

1. Cliquez sur l&#39;icône **[!UICONTROL Enregistrer]** ![](assets/save-icon.png) pour enregistrer votre scénario. Le scénario s’enregistre dans la zone Scénario de Workfront Fusion.

## État du scénario

Les scénarios peuvent être principaux ou inactifs. Vous pouvez modifier l’état du scénario en cliquant sur le bouton Activé/Désactivé dans le détail du scénario.

Pour plus d’informations, voir [Activez ou désactivez un scénario dans [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/activate-or-inactivate-scenario.md) et [Détails du scénario dans [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/scenario-detail.md).

## Planification du scénario

Les scénarios principaux sont exécutés conformément à un planning.

Pour plus d’informations, voir [Planification d’un scénario dans [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/schedule-a-scenario.md).

## Nom du scénario

Pour renommer un scénario, ouvrez-le, cliquez sur son nom dans le coin supérieur gauche et modifiez-le. Appuyez sur Entrée ou cliquez en dehors du champ modifié pour enregistrer le nom du scénario.

## Sélection du premier lot

Vous pouvez sélectionner le premier lot à traiter dans le scénario.

Pour plus d’informations, voir [Sélectionnez l’endroit où commence un module de déclenchement dans [!DNL Adobe Workfront Fusion]](../../workfront-fusion/modules/choose-where-trigger-module-starts.md).

## Nombre de lots renvoyés

Par défaut, les modules renvoient toujours deux lots uniquement. Cela peut être modifié dans les paramètres du module dans le champ . [!UICONTROL Nombre maximal de lots renvoyés].

## Paramètres de scénario avancés

[!DNL Adobe Workfront Fusion] vous donne la possibilité de configurer un certain nombre d’autres paramètres avancés. Pour plus d’informations, voir [Le panneau des paramètres de scénario dans [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/scenario-settings-panel.md).

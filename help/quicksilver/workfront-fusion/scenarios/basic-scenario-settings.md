## Statut du scénario

Les scénarios peuvent être actifs ou inactifs. Vous pouvez modifier le statut du scénario en cliquant sur le bouton « Activé/Désactivé » dans les détails du scénario.

Pour plus d’informations, consultez la section [Activer ou désactiver un scénario dans  [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/activate-or-inactivate-scenario.md) et [Détails du scénario dans  [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/scenario-detail.md).

## Planning du scénario

Les scénarios actifs sont exécutés selon un planning.

Pour plus d’informations, consultez la section [Planifier un scénario dans  [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/schedule-a-scenario.md).

## Nom du scénario

Pour renommer un scénario, ouvrez-le scénario, cliquez sur son nom dans le coin supérieur gauche et modifiez-le. Appuyez sur « Entrée » ou cliquez en dehors du champ édité pour enregistrer le nom du scénario.

## Sélectionnez le premier lot.

Vous pouvez sélectionner le premier lot que vous souhaitez traiter dans le scénario.

Pour plus d’informations, consultez la section [Choisir le point de départ d’un module de déclenchement dans  [!DNL Adobe Workfront Fusion]](../../workfront-fusion/modules/choose-where-trigger-module-starts.md).

## Nombre de lots renvoyés

Par défaut, les modules renvoient toujours deux lots seulement. Ceci peut être modifié dans les paramètres du module dans le champ [!UICONTROL Nombre maximum de lots renvoyés].

## Paramètres avancés du scénario

[!DNL Adobe Workfront Fusion] vous donne la possibilité de configurer un certain nombre d’autres paramètres avancés. Pour plus d’informations, consultez la section [Panneau des paramètres du scénario dans  [!DNL Adobe Workfront Fusion]](../../workfront-fusion/scenarios/scenario-settings-panel.md).

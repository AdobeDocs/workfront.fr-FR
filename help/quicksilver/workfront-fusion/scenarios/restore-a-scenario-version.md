## Restaurer une version de scénario

1. Ouvrez votre scénario.
1. Cliquez sur l’icône [!UICONTROL Plus] ![](assets/more-icon.png), puis sur **[!UICONTROL Versions précédentes]**.

1. Cliquez sur le **[!UICONTROL Sélectionnez la version]**, puis cliquez sur la version que vous souhaitez restaurer.
1. Cliquez sur **[!UICONTROL OK]** :
1. La version de scénario sélectionnée sera restaurée.

>[!NOTE]
>
>La version restaurée du scénario n’est pas enregistrée automatiquement. Si vous souhaitez enregistrer la version restaurée du scénario, vous devez l’enregistrer manuellement.

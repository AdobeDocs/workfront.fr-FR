Vous pouvez modifier les informations de votre [!DNL Adobe Workfront Fusion] profil du compte.

## Modification des paramètres de profil

Pour modifier les informations dans votre profil de compte :

1. Cliquez sur votre avatar au bas du panneau de gauche, puis cliquez sur **[!UICONTROL Profil]**.
1. Cliquez sur : **[!UICONTROL Options générales]**, puis modifiez l’une des options suivantes :

   <table style="table-layout:auto"> 
    <col> 
    <col> 
    <tbody> 
     <tr> 
      <td role="rowheader">[!UICONTROL Organisations]</td> 
      <td> <p>Sélectionnez l’organisation dans laquelle vous souhaitez travailler.<br></p> </td> 
     </tr> 
     <tr> 
      <td role="rowheader">[!UICONTROL Options de fuseau horaire]</td> 
      <td> 
       <ul> 
        <li><strong>[!UICONTROL Web]</strong>: Modifiez le fuseau horaire utilisé pour les dates et heures affichées sur la page [!DNL Workfront Fusion] site web.</li> 
        <li><strong>[!UICONTROL Scénarios]</strong>: Modifiez le fuseau horaire utilisé pour l’analyse et le formatage des dates et heures pendant les exécutions du scénario (lorsque vous ne spécifiez pas explicitement le fuseau horaire). Vous pouvez choisir un fuseau horaire pour chaque paramètre.</li> 
       </ul> </td> 
     </tr> 
     <tr data-mc-conditions=""> 
      <td role="rowheader">[!UICONTROL API]</td> 
      <td> <p>Vous pouvez ajouter un jeton API que vous souhaitez utiliser pour les appels entre [!DNL Workfront Fusion] et une autre application ou service Web. Il s’agit d’une fonctionnalité avancée.</p> </td> 
     </tr> 
    </tbody> 
   </table>

# Explorateur d’API

Vous pouvez voir tous les objets disponibles par le biais de l’API Adobe Workfront, ainsi que leurs champs, dans l’explorateur d’API. Pour afficher l’explorateur API, cliquez [ici](https://developer.adobe.com/workfront/api-explorer/).

Pour plus d’informations sur l’utilisation de l’explorateur d’API, voir [Utiliser l’explorateur d’API](../../wf-api/general/using-api-explorer.md).

# Mise à jour de la gravité des problèmes

Vous pouvez associer une gravité aux problèmes dans Adobe Workfront. Les tâches et les projets n’ont pas de gravité.

Les problèmes sont des événements inattendus qui peuvent empêcher vos projets de se terminer à temps ou dans les limites du budget. Vous pouvez utiliser des statistiques pour indiquer la gravité d’un problème. 

Votre administrateur Workfront définit les statistiques disponibles dans Workfront. Une fois qu’ils ont été créés, vous pouvez les associer à un problème.\
Pour plus d’informations sur la création de statistiques dans Workfront, voir [Création ou personnalisation des gravité de problème](../../../administration-and-setup/customize-workfront/creating-custom-status-and-priority-labels/create-customize-issue-severities.md).

Pour pouvoir mettre à jour sa gravité, vous devez disposer des autorisations de contribution sur un problème. 

Vous pouvez mettre à jour la gravité des problèmes dans les zones suivantes de Workfront :

* Dans le **Modifier le problème** boîte de dialogue
* Dans le **Détails du problème** zone de problème
* Dans une liste de problèmes ou un rapport

## Mise à jour de la gravité des problèmes

Pour mettre à jour la gravité d’un problème dans la zone Détails du problème d’un problème :

1. Accédez au problème dont vous souhaitez mettre à jour la gravité.
1. Cliquez sur **Détails du problème** dans le panneau de gauche.

   Le **Présentation** doit s’afficher par défaut.

1. Cliquez sur le bouton **Gravité** dans le champ **Informations de base** zone.

   ![](assets/nwe-issue-severity-field-in-details-highlighted-350x112.png)

1. Sélectionnez les **Gravité** dans le menu déroulant.

   Les options peuvent varier en fonction de la manière dont votre administrateur Workfront a configuré les statistiques de votre système.

1. Cliquez sur **Enregistrer les modifications**.

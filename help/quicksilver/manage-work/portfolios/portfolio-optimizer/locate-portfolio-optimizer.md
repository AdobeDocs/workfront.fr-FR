## Recherchez la variable [!UICONTROL Portfolio Optimizer]

1. Dans la **[!UICONTROL Menu Principal]**, cliquez sur **[!UICONTROL Portfolios]**.

   Les portefeuilles que vous possédez s’affichent par défaut.

1. (Facultatif) Dans le **[!UICONTROL Filtrer]** dans le menu déroulant, sélectionnez pour afficher un autre ensemble de portefeuilles.
1. Cliquez sur le nom d’un portefeuille pour y accéder.
1. Cliquez sur **[!UICONTROL Optimisation des Portfolios]** dans le panneau de gauche.

   Le [!UICONTROL Portfolio Optimizer] s’affiche.

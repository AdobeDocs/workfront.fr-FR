## Localisez le [!UICONTROL Portfolio Optimizer]

1. Dans le **[!UICONTROL menu principal]**, cliquez sur **[!UICONTROL Portfolios]**.

   Les portefeuilles que vous possédez s’affichent par défaut.

1. (Facultatif) Dans le menu déroulant **[!UICONTROL Filtre]**, sélectionnez pour afficher un autre ensemble de portefeuilles.
1. Cliquez sur le nom d’un portefeuille pour y accéder.
1. Cliquez sur **[!UICONTROL Portfolio Optimization]** dans le panneau de gauche.

   L’ [!UICONTROL Portfolio Optimizer] s’affiche.

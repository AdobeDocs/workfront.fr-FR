## Localiser l’[!UICONTROL optimisateur de portfolio]

1. Dans le **[!UICONTROL Menu principal]**, cliquez sur **[!UICONTROL Portfolios]**.

   Les portfolios que vous possédez s’affichent par défaut.

1. (Facultatif) Dans le menu déroulant **[!UICONTROL Filtrer]**, effectuez une sélection pour afficher un autre jeu de portfolios.
1. Cliquez sur le nom d’un portfolio pour y accéder.
1. Cliquez sur l’**[!UICONTROL optimisateur de portfolio]** dans le panneau de gauche.

   L’[!UICONTROL optimisateur de portfolio] s’affiche.

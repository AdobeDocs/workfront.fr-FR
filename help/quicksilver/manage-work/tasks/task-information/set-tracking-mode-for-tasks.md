# Définir le mode de suivi pour les tâches

Le mode de suivi d’une tâche détermine la manière dont l’état de progression de la tâche est mis à jour dans Adobe Workfront.

Pour plus d’informations sur le mode de suivi sur les tâches, voir [Présentation du mode de suivi des tâches](../../../manage-work/tasks/task-information/task-tracking-mode.md).

## Définir le mode de suivi pour les tâches

Pour définir le mode de tracking :

1. Accédez à la tâche pour laquelle vous souhaitez définir le mode de tracking.
1. Cliquez sur le bouton **Plus** icon ![](assets/qs-more-icon-on-an-object.png)en regard du nom de la tâche, puis cliquez sur **Modifier**.

   La boîte de dialogue Modifier la tâche s’ouvre.

1. Dans le **Paramètres** , utilisez la fonction **Mode de suivi** menu déroulant pour sélectionner le Mode de tracking de la tâche.

   ![](assets/tracking-mode-field-in-edit-task-settings-nwe-350x233.png)

1. Sélectionnez l’une des options suivantes :

   * L’utilisateur doit mettre à jour (il s’agit de l’option par défaut)
   * Heure d’activation de l’hypothèse
   * Ignorer avertissements de retard
   * Fin automatique
   * Tâche antérieure

   Pour plus d’informations sur les options de mode de suivi, voir [Présentation du mode de suivi des tâches](../../../manage-work/tasks/task-information/task-tracking-mode.md)

1. Cliquer sur **Enregistrer**.

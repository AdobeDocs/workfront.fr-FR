# Définir le type de condition d’un projet

En tant que chef de projet, vous pouvez déterminer comment la condition d’un projet est calculée en mettant à jour le type de condition du projet. La condition du projet est une représentation visuelle de la progression du projet.

## Définir le type de condition d’un projet

1. Accédez au projet dont vous souhaitez mettre à jour le type de condition.
1. Utilisez l’une des méthodes suivantes :

   * Cliquez sur le menu **Plus** ![](assets/qs-more-menu.png) à droite du nom du projet, puis sur **Modifier**.
   * Cliquez sur **Détails du projet** dans le panneau de gauche.

   ![](assets/update-condition-type-nwe-350x108.png)

1. Dans le champ **Type de condition**, choisissez l’une des options suivantes :

   * **Manuelle :** le ou la propriétaire du projet définit manuellement la condition du projet.

     Dans ce cas, le ou la propriétaire du projet peut mettre à jour la condition du projet dans l’en-tête du projet ou dans la section Détails du projet.

   * **Statut de la progression :** Workfront définit la condition en fonction du statut de la progression du projet. Pour plus d’informations sur le calcul du statut de la progression, voir [Vue d’ensemble du statut de la progression du projet](../../../manage-work/projects/planning-a-project/project-progress-status.md).

1. Cliquez sur **Enregistrer** lorsque vous modifiez le type de condition dans la zone Modifier le projet.

   Cliquez sur **Enregistrer les modifications** lorsque vous modifiez le type de condition dans la section Détails du projet.

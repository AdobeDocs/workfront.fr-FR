# Vue d’ensemble du bénéfice prévus d‘un projet

<!--Audited: 05/2026-->

Le bénéfice prévu est la somme d’argent estimée dont une organisation pourrait bénéficier si un projet est mené à bien.

Le bénéfice prévu n’est pas calculé par Adobe Workfront. En tant que propriétaire de projet, vous spécifiez manuellement l’avantage prévu d’un projet lorsque vous le modifiez. Nous recommandons à chaque organisation d’établir une politique sur la manière dont cette valeur est déterminée.

Le bénéfice prévu d’un projet affecte la valeur nette d’un projet.

Pour plus d’informations sur le calcul de la valeur nette d’un projet, consultez l’article [Calculer la valeur nette](../../../manage-work/projects/project-finances/calculate-net-value.md).

## Localiser le bénéfice prévu d’un projet

Vous pouvez afficher ou mettre à jour le bénéfice prévu d’un projet dans les zones suivantes de Workfront :

* Dans le récapitulatif du business case. Vous ne pouvez pas modifier l’avantage prévu d’un projet dans le résumé de l’analyse de rentabilité.
Pour plus d’informations sur le business case d’un projet, voir l’article [Créer un business case pour un projet](../../../manage-work/projects/define-a-business-case/create-business-case.md).

* Dans un rapport de projet, lorsque vous ajoutez la colonne Bénéfice prévu à la vue du rapport. Vous pouvez modifier l’avantage prévu d’un projet dans un rapport de projet.\
  Pour plus d’informations sur la création de rapports dans Workfront, voir l’article [Créer un rapport personnalisé](../../../reports-and-dashboards/reports/creating-and-managing-reports/create-custom-report.md).

* Dans l’optimisateur de portfolio. Vous ne pouvez pas modifier l’avantage prévu d’un projet dans Portfolio Optimizer.\
  Pour plus d’informations sur l’optimisateur de portfolio, consultez l’article [Vue d’ensemble de l’optimisateur de portfolio](../../../manage-work/portfolios/portfolio-optimizer/portfolio-optimizer-overview.md).

## Mettre à jour le bénéfice prévu d’un projet

1. Accédez à un projet pour lequel vous souhaitez mettre à jour le bénéfice prévu.
1. Cliquez sur **Détails du projet** dans le panneau de gauche, ou cliquez sur l’icône **Plus** ![](assets/qs-more-icon-on-an-object.png) à droite du nom du projet, puis cliquez sur **Modifier**.
1. Indiquez le montant de l’avantage prévu dans le champ **Avantage prévu**.
1. (Conditionnel) Selon la zone du projet sélectionnée, cliquez sur **Enregistrer les modifications** ou **Enregistrer**.

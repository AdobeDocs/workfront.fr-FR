# Aperçu des avantages prévus du projet

L’avantage prévu est le montant estimé dont pourrait bénéficier une organisation si un projet est terminé.

L’avantage planifié n’est pas calculé par Adobe Workfront. En tant que propriétaire de projet, vous indiquez manuellement l’avantage planifié d’un projet dans l’analyse de cas. Nous recommandons à chaque organisation d’établir une politique sur la manière dont cette valeur est déterminée.

L’avantage prévu d’un projet affecte la valeur nette d’un projet.\
Pour plus d’informations sur le calcul de la valeur nette d’un projet, consultez l’article . [Calculer la valeur nette](../../../manage-work/projects/project-finances/calculate-net-value.md).

## Recherche des avantages prévus d’un projet

Vous pouvez afficher ou mettre à jour les avantages prévus d’un projet dans les zones suivantes de Workfront :

* Dans le résumé des analyses de cas.\
   Pour plus d’informations sur l’analyse de cas d’un projet, reportez-vous à l’article [Création d’une analyse de cas pour un projet](../../../manage-work/projects/define-a-business-case/create-business-case.md).

* Dans un rapport de projet, lorsque vous ajoutez la colonne Bénéfice planifié à la vue du rapport.\
   Pour plus d’informations sur la création de rapports dans Workfront, reportez-vous à l’article [Création d’un rapport personnalisé](../../../reports-and-dashboards/reports/creating-and-managing-reports/create-custom-report.md).

* Dans Portfolio Optimizer.\
   Pour plus d’informations sur Portfolio Optimizer, reportez-vous à l’article [Portfolio Optimizer - Aperçu](../../../manage-work/portfolios/portfolio-optimizer/portfolio-optimizer-overview.md).

## Mise à jour des avantages prévus d’un projet

1. Accédez à un projet pour lequel vous souhaitez mettre à jour l’avantage planifié.
1. Cliquez sur **Analyse de cas** dans le panneau de gauche.
1. Cliquez sur **Modifier les informations du projet** et indiquez le montant de l’avantage prévu dans la variable **Avantage planifié** champ .

1. Cliquer sur **Enregistrer**.
